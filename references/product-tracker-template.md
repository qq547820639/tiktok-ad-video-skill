# 产品追踪与爆款复盘模板 (Product Tracker & Viral Post-Mortem) v2.6

> **重要说明**：本模板是一个**可选的数据记录工具**，而非必须完成的作业。
> 
> **Skill 的自我迭代主要依赖你在对话中反馈的数据**——每次任务结束后，Skill 会主动询问视频表现，并根据你的回复自动分析成败原因、调整后续生成策略。
> 
> 本模板适用于以下场景：
> - 你想手动追踪多个产品的长期表现
> - 你需要生成周报/月报向团队汇报
> - 你想积累数据用于更复杂的自定义分析
> 
> **如果你只是日常使用 Skill，无需填写本模板，直接在对话中告诉 Skill 数据即可。**

> **更新版本**：v2.6 (2026.05) —— 统一版本号，新增原生感启用、Fast模式成本、社交货币分享类型等字段，对齐 v2.6 评估体系。

## 四、爆款归因分析 (Viral Post-Mortem)

> **填写时机**：视频发布 7 天后，或播放量达到显著层级（如 100K+ 或低于 1K）时填写。

### 整体表现判定

| 播放量层级 | 判定结果 | 核心结论 |
| :--- | :--- | :--- |
| 1M+ | 大爆款 ✅ | |
| 100K - 1M | 小爆款 ✅ | |
| 10K - 100K | 中等表现 ⚠️ | |
| 1K - 10K | 表现不佳 ❌ | |
| <1K | 限流/内容问题 ❌ | |

### 归因分析表

| 分析维度 | 成功因素 (若爆) / 失败原因 (若扑) | 证据来源 |
| :--- | :--- | :--- |
| **钩子有效性** | [前 3 秒是否触发留存？钩子类型是否与产品匹配？] | 完播率曲线 / 评论反馈 |
| **品类-模板匹配度** | [功能效果型产品是否使用了对应的多镜头模板？] | 完播率分段数据 |
| **五维架构执行** | [技术基底、镜头运动、光影系统等维度是否执行到位？] | 视觉评估 / 用户反馈 |
| **原生感效果** | [UGC 风格是否增强了信任感？是否有广告抵触评论？] | 评论情感分析 / 转化数据 |
| **TikTok 测试池突破** | [是否成功突破 200-500 人测试池？前 3 秒完播率是否 >70%？] | 播放量增长曲线 |
| **YouTube 划走率** | [划走率是否低于平台平均？前 2 秒是否抓住注意力？] | Shorts 分析面板 |
| **收藏引导效果** | [收藏率是否 >10%？引导话术是否有效？] | 收藏数据 |

| **视觉呈现** | [画面冲击力、运镜质感、AI 瑕疵] | 质量评估得分 / 用户评论 |

| **竞品环境** | [同期是否有同类爆款分流？] | 手动搜索观察 |

## 使用说明

1. **日常使用**：直接在对话中向 Skill 反馈视频表现数据（播放量、完播率、收藏率、分享率），Skill 会自动记录并迭代。
2. **手动记录（可选）**：如需周报或深度分析，可复制本模板填写。
3. **迭代触发**：Skill 会在每次收到你的数据反馈后自动更新内部权重和策略，无需你手动干预。
