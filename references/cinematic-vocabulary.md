## Prompt Length Guide

| Element | Recommended Length |
|---------|-------------------|
| Full prompt | 150–300 words (sweet spot for Seedance 2.0) |
| Hook description | 2–3 sentences |
| Product demo | 3–5 sentences |
| Price/brand overlay | 2–3 sentences |
| CTA | 1–2 sentences |
| Safety zone + language rules | 2 fixed sentences (always include) |

**Too short** (<100 words): Seedance fills in blanks with generic content
**Too long** (>400 words): Model may ignore later instructions or produce incoherent results
**Sweet spot** (150–300 words): Enough detail for cinematic quality, short enough for consistent execution

## Viral Boost Pack — High-CTR Prompt Enhancers (爆款专属提示词强化包)

> **Instruction**: When generating Seedance 2.0 video prompts, include at least 2 terms from Section A at the beginning of the prompt, and at least 1 term from Section B to suppress common AI artifacts.

### A. Viral Prefix Keywords (Prompt Opening)
*Include 2+ of these at the start of every prompt to activate high-CTR visual style.*

| Keyword | English Term (use this in prompt) | Effect |
| :--- | :--- | :--- |
| 视觉冲击 | `Visual impact`, `Eye-catching` | Boosts first-frame attractiveness |
| 高质感广告大片 | `High-end commercial look` | Elevates production value perception |
| 电影级微距 | `Cinematic macro shot` | Enhances detail texture quality |
| 病毒式美学 | `Viral TikTok aesthetic` | Aligns output with algorithm-preferred style |
| 丝滑运镜 | `Smooth camera motion` | Improves viewing flow |
| 高点击率风格 | `High CTR style` | General viral quality boost |
| 干脆利落 | `Snappy motion` | Prevents sluggish pacing |

**Recommended combo** (copy-paste as prompt opening):
```

```

### B. Anti-AI Artifact Instructions (去 AI 味指令)
*Include 1-2 of these to suppress common Seedance artifacts: empty pans, unnatural slowness, robotic feel.*

| Problem It Solves | English Prompt Instruction |
| :--- | :--- |
| Slow empty panning shots | `Avoid slow cinematic pans` |
| Sluggish pace | `Snappy motion, Quick cuts` |
| Robotic/stabilizer feel | `Handheld POV shot, Organic unscripted feel` |
| No visual closure | `Freeze frame on the perfect result for 2 seconds` |
| Text gets covered by UI | `Clear space for text overlay in center 60%` |
| Overly polished look | `Authentic lived-in feel, Natural imperfections` |

### C. Platform-Specific Suffix (append at prompt end)
*Choose one based on target distribution. Default to "Universal" for multi-platform.*

| Platform | Suffix (append to prompt end) | Purpose |
| :--- | :--- | :--- |
| **Universal (All)** | `9:16 vertical, 15 seconds loop-ready, sharp focus on product` | Ensures format correctness |
| **Pinterest** | `Self-explanatory visual, bold text overlay space, muted playback friendly` | Optimizes for silent autoplay |
| **TikTok/Reels** | `End with a seamless loop transition` | Boosts completion + replay rate |

### D. Negative Prompt Reference (Avoid These)
*If the platform supports negative prompts, use these to suppress common failures:*

| Unwanted Effect | Negative Prompt Terms |
| :--- | :--- |
| Warping/morphing | `Warping, Morphing, Jittery animation` |
| Face/hand distortion | `Deformed face, Disfigured hands` |
| Random text appearing | `Text overlays, Watermarks, Logos in frame` |
| Pacing too slow | `Slow motion, Time-lapse, Long empty pauses` |
| Generic stock footage feel | `Stock footage, Generic b-roll, Template animation` |
