# 视频质量与爆款潜力评分表 (Evaluation Rubric) v2.12

> **用途**：对 Seedance 2.0 生成的视频进行标准化评估，并为 Prompt 文本预评估提供评分标准，实现“事前+事后”双重质量保障。
> **评分原则**：
> - **Prompt 文本预评估**：满分 100 分，四档评级（优秀/合格/需小修/需大修），含非线性惩罚。
> - **直给型广告视频评估**：满分 100 分，≥75 分可发布。
> - **叙事型软广视频评估**：满分 145 分（五维 100 分 + 叙事专项 45 分），≥110 分可发布。
> **更新版本**：v2.12 (2026.05) —— 优化 Prompt 预评估分档阈值，增加非线性惩罚规则，使迭代方向更明确。


## 一、Prompt 文本质量评分表（提交前预评估）

> **用途**：在提交即梦 AI 生成视频**之前**，对 Prompt 文本进行预评估，从源头把控创意质量，避免因 Prompt 质量不足导致的积分浪费。
> **评分原则**：满分 100 分，四档评级，含非线性惩罚。

### 1.1 评分总览

| 维度 | 分值 | 评估重点 |
| :--- | :--- | :--- |
| **前 3 秒钩子强度** | 30 分 | 是否包含高冲击力的视觉冲突、悬念或认知失调？声音钩子是否明确？口播时机是否正确？ |
| **声音策略明确度** | 25 分 | 是否明确标注声音策略（ASMR/音效/环境音）？音频同步指令是否精准？口播时机是否指定？ |
| **垂直领域信号** | 20 分 | 是否在前 5 秒内口述+字幕双重强化核心主题词？是否预留醒目大字报空间？ |
| **原生感设计** | 15 分 | 是否使用原生感关键词？是否避免了精修广告词汇？ |
| **去 AI 味指令完整性** | 10 分 | 是否包含防畸变指令（手部/面部）？是否避免了负面词表中的词汇？ |
| **总分** | **100 分** | 四档评级见下表 |

### 1.2 详细评分标准

#### 1.2.1 前 3 秒钩子强度 (30 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 26-30 分 | 首帧明确包含“认知失调”或“视觉奇观”元素；**明确标注前 3 秒声音策略**（如“前3秒纯 ASMR 煎蛋声”）；口播第 3 秒后才进入；用户看完 3 秒会产生明确好奇心 |
| 18-25 分 | 有视觉冲击或悬念，但声音钩子不够突出；或口播进入略早（第 2 秒进入） |
| 10-17 分 | 画面平淡，无明确声音钩子设计；口播第 1 秒即进入抢占注意力 |
| 0-9 分 | 前 3 秒为空镜、黑屏、静态场景，或完全无声/仅有平淡人声 |

#### 1.2.2 声音策略明确度 (25 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 22-25 分 | Prompt 中**明确标注声音策略**（如 `前3秒纯 ASMR 音效`）；音频同步指令精准（如 `Visual beats sync with heavy bass drops`）；口播进入时机明确 |
| 15-21 分 | 有声音描述但不够具体；或缺少音频同步指令 |
| 8-14 分 | 声音描写模糊（如“有声音”），无具体策略 |
| 0-7 分 | 无任何声音相关指令 |

#### 1.2.3 垂直领域信号 (20 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 18-20 分 | Prompt 中明确要求**前 5 秒内口述+字幕双重强化核心主题词**；预留醒目大字报空间（如 `bold text overlay space`） |
| 12-17 分 | 有主题词出现，但仅口述或仅字幕，未双重强化；或大字报空间未明确预留 |
| 6-11 分 | 主题词出现时间超过前 5 秒，或不明显 |
| 0-5 分 | 无任何垂直领域信号设计 |

#### 1.2.4 原生感设计 (15 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 13-15 分 | 使用原生感关键词：`Authentic unscripted reaction`, `Lived-in messy background`, `Natural window light`, `Handheld POV`；明确避免精修广告词汇 |
| 9-12 分 | 部分使用原生感词汇，但仍混有影棚/精修描述 |
| 4-8 分 | 高度精修广告风格描述，缺乏 UGC 感 |
| 0-3 分 | 完全未考虑原生感，纯精修广告文案 |

#### 1.2.5 去 AI 味指令完整性 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 包含完整防畸变指令：`Well-formed hands`, `No extra fingers`, `No face distortion`, `Stable object geometry`；已避免负面词表词汇（`Warping`, `Morphing` 等） |
| 6-8 分 | 有防畸变指令但不完整 |
| 3-5 分 | 仅有个别指令，遗漏关键项（如手部防畸变） |
| 0-2 分 | 无任何去 AI 味指令，或包含负面词表词汇 |

### 1.3 预评估结果分档与执行动作

> **优化说明**：将原先简单的二值判断细化为四档评级，并增加针对致命低分项的惩罚机制。

| 总分区间 | 评级 | 执行动作 |
| :--- | :--- | :--- |
| **≥ 85 分** | ✅ 优秀 | 可直接提交即梦生成视频，预期高概率（>85%）通过后续视频质量评估。 |
| **80-84 分** | ⚠️ 合格 | 允许提交，但建议根据低分项建议进行小修后提交，可进一步提升视频质量。 |
| **70-79 分** | 🔄 需小修 | 不合格，必须退回阶段 2 或阶段 3 优化低分项。通常无需重新进行钩子盲选，微调脚本和 Prompt 即可。 |
| **< 70 分** | ❌ 需大修 | 不合格，建议从阶段 1 重新评估钩子方向，或检查产品品类与钩子选型是否匹配。 |

> **非线性惩罚规则**：若以下任一“致命项”得分低于 50 分（满分 100 分制下），无论总分多少，评级自动降一档（优秀→合格，合格→需小修，需小修→需大修）：
> - **前 3 秒钩子强度 (H)**：低于 50 分表示钩子基本失效，必须优先修复。
> - **声音策略明确度 (A)**：低于 50 分表示声音设计严重缺失，影响前 3 秒穿透力。

**示例**：
- 某 Prompt 总分 86 分，但 H=45 分。因触发非线性惩罚，评级从“优秀”降为“合格”，需优化钩子后方可提交。
- 某 Prompt 总分 73 分，A=48 分。因触发非线性惩罚，评级从“需小修”降为“需大修”，建议重新评估钩子方向。

### 1.4 预评估执行模板

**输出给用户（通过示例）**：
```

- 垂直领域信号：18/20（✅ 合格）
- 原生感设计：13/15（✅ 合格）
- 去AI味指令完整性：9/10（✅ 合格）
- 总分：89/100（评级：✅ 优秀）

可直接提交即梦生成视频。建议优化项：[具体建议]

```


## 二、直给型广告视频评分总览（生成后评估）

| 维度 | 分值 | 评估重点 |
| :--- | :--- | :--- |
| **技术质量** | 20 分 | 画面清晰度、运镜质感、AI 生成瑕疵 |
| **爆款钩子** | 30 分 | 前 3 秒冲击力（含声音钩子）、完播率潜力、复播引导设计 |
| **平台适配** | 20 分 | 收藏/分享引导力、垂直领域信号、TikTok SEO、YouTube 划走率控制、Meta 真实价值感、静音可读性 |

| **算法信号** | 15 分 | 复播率预估、收藏引导力预估、分享引导力预估 |

## 三、叙事型软广视频评分总览（生成后评估）

叙事型最终得分 = **直给型五维分（100 分制） + 叙事专项分（45 分制）**，满分 **145 分**。

**建议发布阈值**：
- ≥ 110 分：可发布
- 90-109 分：微调后发布
- < 90 分：重新构思

### 3.1 叙事专项评分细则（45 分）

| 指标 | 分值 | 评估标准 |
| :--- | :--- | :--- |
| **悬念维持度** | 10 分 | 前 12 秒是否建立困境？12-22 秒悬念是否持续增强？观众是否有理由看到最后？ |
| **情绪弧线完整度** | 10 分 | 是否有清晰的情绪变化轨迹（疲惫→好奇→惊讶→治愈）？情绪转折是否自然？ |
| **产品植入自然度** | 10 分 | 产品是否作为“故事道具”出现而非“广告主角”？是否无硬广痕迹？ |
| **评论区引导力** | 10 分 | 视频内容是否自然激发“WHAT IS THIS?”类评论？悬念是否驱动用户留言？ |
| **零提及纪律** | 5 分 | 视频内是否严格避免品牌名/Logo/价格/购买引导语？ |

## 四、失败模式及修复方案速查表

| 失败表现 | 可能原因 | 修复方案 |
| :--- | :--- | :--- |
| **Prompt 预评估 < 70 分（需大修）** | 钩子方向错误或品类匹配失败 | 退回阶段 1，对照品类选型表重新选择钩子 |
| **Prompt 预评估 70-79 分（需小修）** | 钩子方向正确但执行细节不足 | 退回阶段 2/3，微调脚本和 Prompt，强化低分项 |
| **钩子强度 H < 50 分** | 静态场景或无冲突开头 | 必须增加动态冲突或奇观展示，明确前3秒声音钩子 |
| **声音策略 A < 50 分** | 无声音设计或口播过早抢占 | 明确前3秒纯音效，口播延后至第3秒 |
| 前 3 秒平淡 | 钩子类型选错或未强调开场 | 对照品类选型表换钩子；前3秒增加 ASMR/音效 |
| **播放量卡在 200-500** | 未通过测试池，前3秒留存不足 | 强化前3秒声音钩子+视觉冲击；美国市场考虑切换到叙事型 |
| **垂直领域信号缺失** | 算法无法精准识别内容领域 | 前5秒内口述+字幕双重出现核心主题词 |
| 完播率低，中间划走 | 信息密度不足，有空镜 | 增加指令：`Snappy motion, No empty frames` |
| 收藏率低 | 未强调实用价值或下次使用 | 增加引导语：“Save for your next ___” + 收藏图标动画 |
| 分享率低 | 未赋予社交货币 | 对照品类更换社交货币类型 |
| YouTube Shorts 推荐量低 | 前 2 秒划走率高 | 前2秒声音钩子+极端特写，添加 `Snappy motion, no slow pans` |
| Meta 限流 | 内容过于病毒式煽动 | 启用原生感策略，减少夸张前缀 |
| AIGC 标签未勾选 | 发布时未提醒合规操作 | 发布前强制提醒勾选 AI 标签 |
| **叙事型悬念不足** | 中间段落拖沓 | 剪掉冗余镜头，确保每 10 秒有一个“小钩子” |
| **叙事型评论少** | 视频未激发自然追问 | 增加“Mystery Object”元素，强化评论区运营 |
| **叙事型零提及违规** | 视频内出现品牌/价格 | 重新生成违规段落，确保全程零提及 |
