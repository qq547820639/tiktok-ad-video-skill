# 视频质量与爆款潜力评分表 (Evaluation Rubric) v2.10

> **用途**：对 Seedance 2.0 生成的视频进行标准化评估，判断是否达到发布标准，并为迭代提供数据依据。
> **评分原则**：直给型广告满分 100 分（五大维度）；叙事型软广满分 125 分（五大维度 + 叙事专项）。
> **更新版本**：v2.10 (2026.05) —— 新增叙事型内容专项评估（5 项，45 分），区分直给型与叙事型评分标准。


## 一、评分总览（直给型）

| 维度 | 分值 | 评估重点 |
| :--- | :--- | :--- |
| **技术质量** | 20 分 | 画面清晰度、运镜质感、AI 生成瑕疵 |
| **爆款钩子** | 30 分 | 前 3 秒冲击力（含声音钩子）、完播率潜力、复播引导设计 |
| **平台适配** | 20 分 | 收藏/分享引导力、垂直领域信号、TikTok SEO、YouTube 划走率控制、Meta 真实价值感、静音可读性 |
| **导演执行** | 15 分 | 五维架构执行度、原生感执行度、声音钩子执行度、垂直领域信号执行度、多镜头结构完整性、音频同步效果、转场质量 |
| **算法信号** | 15 分 | 复播率预估、收藏引导力预估、分享引导力预估 |
| **总分** | **100 分** | ≥75 分可发布；60-74 分需优化；<60 分废弃重做 |


## 二、技术质量评分细则 (20 分)

### 2.1 画面清晰度与分辨率 (6 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 6 分 | 画面锐利，1080x1920 满分辨率，无模糊、马赛克、噪点 |
| 4-5 分 | 轻微柔化但主体清晰，分辨率达标 |
| 2-3 分 | 局部模糊，边缘有锯齿，影响观感 |
| 0-1 分 | 大面积模糊、变形、画质低劣，不可用 |

### 2.2 运镜质感与构图 (6 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 6 分 | 运镜丝滑自然，镜头运动（推拉摇移跟）执行精准，主体始终处于视觉中心，构图符合 9:16 竖屏美学 |
| 4-5 分 | 轻微抖动但不影响观看，运镜意图可辨但不够精准 |
| 2-3 分 | 晃动明显或构图失衡，运镜与内容不匹配 |
| 0-1 分 | 剧烈抖动、眩晕感、主体出画，或完全无运镜设计 |

### 2.3 AI 生成瑕疵控制 (8 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 8 分 | 无人物手脸变形、物体形变、鬼影、闪烁等 AI 常见瑕疵 |
| 5-7 分 | 轻微瑕疵但不被普通观众察觉（如指尖略糊） |
| 3-4 分 | 明显瑕疵引起注意（如手指数量异常、物体融化解体） |
| 0-2 分 | 严重畸变、恐怖谷效应，直接劝退观众 |


## 三、爆款钩子评分细则 (30 分)

### 3.1 前 3 秒留存预测 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 首帧即含“认知失调”或“视觉奇观”，**且前 3 秒有明确的声音钩子（ASMR/音效/环境音）**，口播在第 3 秒后才进入，让人本能停止划动 |
| 6-8 分 | 有明显视觉冲击或悬念，但声音钩子不够突出或口播进入过早 |
| 3-5 分 | 平淡开场，无声音钩子，需要观众耐心等待信息 |
| 0-2 分 | 前 3 秒完全是空镜、黑屏、无关画面，必划走 |

### 3.2 完播率潜力 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 15 秒内信息密度高，无冗余帧，每 3 秒有“小钩子”，结尾有定格引导 |
| 6-8 分 | 整体流畅但有 1-2 秒拖沓，结尾无明显停留 |
| 3-5 分 | 节奏松散，中间有无效空镜，结尾仓促 |
| 0-2 分 | 大量无效时长，观众无法坚持看完 |

### 3.3 复播引导设计 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 内置明确复播彩蛋（信息差、节奏中断、双重结局），观众有明显回看动机 |
| 6-8 分 | 有一定复播引导但不够突出或自然 |
| 3-5 分 | 无意间可能触发复播，但无主动设计 |
| 0-2 分 | 无任何复播引导，一次看完即走 |


## 四、平台适配评分细则 (20 分)

### 4.1 收藏/分享引导力 (6 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 5-6 分 | 明确引导**收藏**和**社交货币分享**，理由充分（实用价值锚定、展现品味、圈层归属），话术与视觉配合到位 |
| 3-4 分 | 有引导但理由不够充分或只有其中一项 |
| 1-2 分 | 仅有浅层引导（如“点赞关注”），未涉及收藏/分享 |
| 0 分 | 无任何收藏/分享引导 |

### 4.2 TikTok 深度互动与垂直领域适配 (4 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 4 分 | 内容具备评论/分享动机，结尾有明确深度评论引导；**前 5 秒口述+字幕双重强化核心主题词**，钩子类型符合品类选型对照表（未选错钩子）；前 3 秒冲击力足以突破 200-500 人测试池 |
| 2-3 分 | 有一定讨论价值，但引导不明确或偏浅层；垂直领域信号存在但不够突出；钩子类型与品类匹配度一般 |
| 1 分 | 内容看完即走，无评论/分享理由；无垂直领域信号；钩子类型选错 |
| 0 分 | 内容封闭，无互动可能 |

### 4.3 YouTube Shorts 适配 (4 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 4 分 | 视频画面中包含可搜索的关键词视觉元素；前 2 秒声音钩子强，预估划走率 <30%；标题/描述 SEO 潜力高；有引导多平台分享的提示 |
| 2-3 分 | 内容可被搜索但关键词不够突出，或前 2 秒有划走风险（30-50%） |
| 1 分 | 内容泛化，难以匹配具体搜索意图；前 2 秒平淡 |
| 0 分 | 纯氛围内容，无搜索价值；划走率预估 >50% |

### 4.4 Meta Reels 适配 (3 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 3 分 | 内容传递真实价值，前 5 秒亮明品牌，符合 UTIS 一致性，无空洞煽动；采用原生感策略 |
| 2 分 | 有价值输出但品牌露出不足或风格略浮夸 |
| 1 分 | 偏病毒式娱乐，真实价值感弱 |
| 0 分 | 空洞口号、过度夸张、无实质信息 |

### 4.5 Pinterest 适配 (3 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 3 分 | 静音状态下完全可懂，画面有大号文字叠加，封面帧具备高点击率；建议多格式发布 |
| 1-2 分 | 静音可懂但文字偏小或不突出，或依赖音频 |
| 0 分 | 无音频则完全不知所云 |


## 五、导演执行评分细则 (15 分)

### 5.1 五维架构执行度 (2 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 2 分 | 五维（技术基底/镜头运动/视觉元素/光影系统/动态设计）均明确且执行到位，画面具有电影感 |
| 1 分 | 缺少 1-2 个维度或执行不够精准，整体效果尚可 |
| 0 分 | 仅依赖描述性语言，未使用结构化指令，画面平淡 |

### 5.2 原生感执行度 (2 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 2 分 | 视频风格符合 UGC 特征：素人演员感、生活化场景、自然光线、手持自拍感、轻微瑕疵可见，无明显广告精致感 |
| 1 分 | 部分符合原生感特征，但仍有明显摆拍或影棚痕迹 |
| 0 分 | 高度精修广告风格，与原生感策略相悖（除非产品需要） |

### 5.3 声音钩子执行度 (3 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 3 分 | 前 3 秒有明确的 ASMR/音效/环境音作为声音钩子，口播第 3 秒后才进入；声音与画面内容高度配合，穿透力强 |
| 2 分 | 有声音钩子但不够突出，或口播进入略早（第 2 秒进入） |
| 1 分 | 声音钩子与画面关联弱，或口播第 1 秒即进入抢占注意力 |
| 0 分 | 前 3 秒无声或仅有平淡人声，无任何声音钩子设计 |

### 5.4 垂直领域信号执行度 (2 分)

| 得分 | 标准描述 |
| :--- | :--- |

| 1 分 | 有主题词出现，但仅口述或仅字幕，未双重强化 |
| 0 分 | 主题词出现时间超过前 5 秒，或不明显，或无任何垂直领域信号 |

### 5.5 多镜头结构完整性 (2 分)

| 得分 | 标准描述 |
| :--- | :--- |

## 八、叙事型内容专项评估（v2.10 新增）

> **适用场景**：仅对 45-60 秒叙事型软广启用。直给型广告跳过本章节。

### 8.1 评分总览（叙事型）

叙事型最终得分 = **直给型五维分（100 分制） + 叙事专项分（45 分制）**，满分 **145 分**。

**建议发布阈值**：
- ≥ 110 分：可发布
- 90-109 分：微调后发布
- < 90 分：重新构思

### 8.2 叙事专项评分细则（45 分）

#### 8.2.1 悬念维持度 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 前 12 秒清晰建立困境/好奇心；12-22 秒悬念持续增强；观众有强烈动机看到最后（产品揭晓时刻）；全片无无聊段落 |
| 6-8 分 | 有悬念建立，但中间部分段落节奏略拖沓；观众可能在第 30 秒左右流失 |
| 3-5 分 | 悬念不够突出，或中间大段平淡内容；观众难以坚持到产品揭晓 |
| 0-2 分 | 平铺直叙，无悬念设计；观众在 15 秒内大量划走 |

#### 8.2.2 情绪弧线完整度 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 有清晰的情绪变化轨迹（如疲惫→好奇→惊讶→治愈）；情绪转折自然流畅；结尾有明确的情绪释放 |
| 6-8 分 | 有情绪变化，但转折略显生硬或缺少某一环节；结尾情绪收束不够到位 |
| 3-5 分 | 情绪单一，无明显变化；或情绪转折突兀，观众出戏 |
| 0-2 分 | 无情绪设计，纯粹产品展示或功能罗列 |

#### 8.2.3 产品植入自然度 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 产品作为“故事道具”自然出现，而非“广告主角”；观众在无意识中记住产品效果；无任何硬广痕迹 |
| 6-8 分 | 产品植入略显刻意，但不影响整体故事观感；观众能感觉到“这是在卖东西”但不反感 |
| 3-5 分 | 产品植入生硬，打断叙事节奏；广告痕迹明显 |
| 0-2 分 | 纯粹广告式展示，与叙事脱节；观众立刻识别为广告并划走 |

#### 8.2.4 评论区引导力 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 视频内容自然激发“WHAT IS THIS?”类好奇心评论；悬念设计直接驱动用户留言追问产品信息 |
| 6-8 分 | 有一定评论引导力，但需要评论区运营辅助才能激发追问 |
| 3-5 分 | 评论主要依赖运营小号带动，视频本身缺乏自然追问动机 |
| 0-2 分 | 评论区冷淡，用户看完即走，无任何追问欲望 |

#### 8.2.5 零提及纪律 (5 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 5 分 | 视频内严格避免：品牌名、Logo、价格、购买引导语（link/buy/shop/order）、购买页面截图 |
| 3-4 分 | 视频内出现 1 处轻微违规（如品牌名一闪而过），但不影响整体“软广”感知 |
| 1-2 分 | 出现多处违规，或关键处有明显的硬广痕迹 |
| 0 分 | 视频内直接喊出价格/购买引导，完全违背“零提及”原则 |

| **垂直领域信号缺失** | 算法无法精准识别内容领域 | 前5秒内口述+字幕双重出现核心主题词 |
