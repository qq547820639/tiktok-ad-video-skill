# 视频质量与爆款潜力评分表 (Evaluation Rubric) v2.15.1

> **用途**：对 Seedance 2.0 生成的视频进行标准化评估，并为 Prompt 文本预评估提供评分标准，实现“事前+事后”双重质量保障。
> **评分原则**：
> - **AI 自动执行时**：优先使用 `SKILL.md` §7 的 **「三层快速诊断」**（致命层/核心层/卫生层），核心层已量化。
> - **人工复盘时**：使用本文档的 **「Prompt 文本质量评分表」**（五维度 100 分制）和 **「视频成片评估表」**（直给型 100 分/叙事型 145 分）进行深度分析。
> **更新版本**：v2.15.1 (2026.05) —— 核心层检查项增加量化阈值，减少主观判断。


## 一、AI 自动执行：三层快速诊断（优先使用 · v2.15.1 量化版）

> **说明**：此逻辑已内置在 `SKILL.md` §7 中。AI 在阶段 3.5 应优先使用三层诊断，本文档的五维评分表用于人工复盘时参考。

| 层级 | 检查项 | 量化标准 | 不通过动作 |
| :--- | :--- | :--- | :--- |
| 🔴 **致命层** | 钩子是否有视觉冲突？是否有声音钩子？ | 二选一：任一不满足即不通过 | **禁止提交**，退回阶段 2 |
| 🟡 **核心层** | 垂直信号是否前 5 秒口述+字幕双出现？原生感是否启用？ | **垂直信号缺失扣 5 分**；**原生感未启用扣 3 分**。累计扣分 ≥5 分即不通过 | **警告 + 微调**，用户确认后提交 |
| 🟢 **卫生层** | 是否含否定式负向词？是否超过 100 词/段？ | 含否定词自动替换为正向锚点词；超长自动拆分为独立段落 | **自动修正**，直接提交 |

### 1.1 核心层量化细则

| 检查项 | 判定标准 | 扣分值 | 说明 |
| :--- | :--- | :--- | :--- |
| **垂直领域信号** | 前 5 秒内口述+字幕**同时出现**核心主题词 | 缺失扣 **5 分** | 仅口述或仅字幕视为“部分缺失”，扣 3 分 |
| **原生感策略** | 提示词中包含原生感关键词（如 `Natural window light`, `Handheld POV`, `Authentic unscripted feel`） | 未启用扣 **3 分** | 若为奢侈品/精修广告场景，用户明确要求可不扣分 |

> **累计扣分 ≥5 分** → 核心层不通过，需警告 + 微调后重新诊断。

### 1.2 与五维评分表的对应关系

三层诊断是五维评分表的**简化执行版**，对应关系如下：

| 三层诊断 | 对应五维维度 | 简化方式 |
| :--- | :--- | :--- |
| 🔴 致命层 | 前 3 秒钩子强度 (30分) + 声音策略明确度 (25分) | 合并为“是否有视觉冲突+是否有声音钩子” |
| 🟡 核心层 | 垂直领域信号 (20分) + 原生感设计 (15分) | 合并为“垂直信号是否双出现+原生感是否启用”，增加量化阈值 |
| 🟢 卫生层 | 去 AI 味指令完整性 (10分) + 提示词长度控制 | 合并为“是否含否定词+是否超长” |


## 二、Prompt 文本质量评分表（人工复盘用）

> **用途**：在提交即梦 AI 生成视频**之前**，对 Prompt 文本进行深度预评估。日常使用优先走三层诊断，本表用于每周复盘或校准分析。

### 2.1 评分总览

| 维度 | 分值 | 评估重点 |
| :--- | :--- | :--- |
| **前 3 秒钩子强度** | 30 分 | 是否包含高冲击力的视觉冲突、悬念或认知失调？声音钩子是否明确？口播时机是否正确？ |
| **声音策略明确度** | 25 分 | 是否明确标注声音策略（ASMR/音效/环境音）？音频同步指令是否精准？口播时机是否指定？ |
| **垂直领域信号** | 20 分 | 是否在前 5 秒内口述+字幕双重强化核心主题词？是否预留醒目大字报空间？ |
| **原生感设计** | 15 分 | 是否使用原生感关键词？是否避免了精修广告词汇？ |
| **去 AI 味指令完整性** | 10 分 | 是否使用正向锚点词（非否定式负向词）？是否包含防畸变指令？ |
| **总分** | **100 分** | 四档评级见下表 |

### 2.2 详细评分标准

#### 2.2.1 前 3 秒钩子强度 (30 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 26-30 分 | 首帧明确包含“认知失调”或“视觉奇观”元素；**明确标注前 3 秒声音策略**；口播第 3 秒后才进入；用户看完 3 秒会产生明确好奇心 |
| 18-25 分 | 有视觉冲击或悬念，但声音钩子不够突出；或口播进入略早（第 2 秒进入） |
| 10-17 分 | 画面平淡，无明确声音钩子设计；口播第 1 秒即进入抢占注意力 |
| 0-9 分 | 前 3 秒为空镜、黑屏、静态场景，或完全无声/仅有平淡人声 |

#### 2.2.2 声音策略明确度 (25 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 22-25 分 | 明确标注声音策略（如 `前3秒纯 ASMR 音效`）；音频同步指令精准；口播进入时机明确 |
| 15-21 分 | 有声音描述但不够具体；或缺少音频同步指令 |
| 8-14 分 | 声音描写模糊（如“有声音”），无具体策略 |
| 0-7 分 | 无任何声音相关指令 |

#### 2.2.3 垂直领域信号 (20 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 18-20 分 | 明确要求**前 5 秒内口述+字幕双重强化核心主题词**；预留醒目大字报空间；SEO 关键词植入合理 |
| 12-17 分 | 有主题词出现，但仅口述或仅字幕，未双重强化；或大字报空间未明确预留 |
| 6-11 分 | 主题词出现时间超过前 5 秒，或不明显 |
| 0-5 分 | 无任何垂直领域信号设计 |

#### 2.2.4 原生感设计 (15 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 13-15 分 | 使用原生感关键词：`Authentic unscripted reaction`, `Lived-in messy background`, `Natural window light`, `Handheld POV`；明确避免精修广告词汇 |
| 9-12 分 | 部分使用原生感词汇，但仍混有影棚/精修描述 |
| 4-8 分 | 高度精修广告风格描述，缺乏 UGC 感 |
| 0-3 分 | 完全未考虑原生感，纯精修广告文案 |

#### 2.2.5 去 AI 味指令完整性 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | **全部使用正向锚点词**（`Well-formed hands`, `Stable object geometry`, `Crystal-clear`, `Steady motion`）；无任何否定式负向词 |
| 6-8 分 | 大部分使用正向词汇，偶有 1 个否定式负向词 |
| 3-5 分 | 部分使用正向词汇，但混有多个否定式负向词（如 `No extra fingers`, `No warping`） |
| 0-2 分 | 大量使用否定式负向词，或缺失关键防畸变指令 |

### 2.3 预评估结果分档

| 总分区间 | 评级 | 执行动作 |
| :--- | :--- | :--- |
| **≥ 85 分** | ✅ 优秀 | 可直接提交即梦生成视频 |
| **80-84 分** | ⚠️ 合格 | 允许提交，建议微调低分项 |
| **70-79 分** | 🔄 需小修 | 不合格，必须退回阶段 2/3 优化 |
| **< 70 分** | ❌ 需大修 | 不合格，建议从阶段 1 重新选择钩子方向 |

> **非线性惩罚**：若前 3 秒钩子强度(H)或声音策略明确度(A)得分低于 50 分（满分 100 分制下），无论总分多少，评级自动降一档。


## 三、直给型广告视频评分总览（生成后评估）

| 维度 | 分值 | 评估重点 |
| :--- | :--- | :--- |
| **技术质量** | 20 分 | 画面清晰度、运镜质感、AI 生成瑕疵 |
| **爆款钩子** | 30 分 | 前 3 秒冲击力（含声音钩子）、完播率潜力、复播引导设计 |
| **平台适配** | 20 分 | 收藏/分享引导力、垂直领域信号、TikTok SEO、YouTube 划走率控制、Meta 真实价值感、静音可读性 |
| **导演执行** | 15 分 | 五维架构执行度、原生感执行度、声音钩子执行度、垂直领域信号执行度、多镜头结构完整性、音频同步效果、转场质量 |
| **算法信号** | 15 分 | 复播率预估、收藏引导力预估、分享引导力预估 |
| **总分** | **100 分** | ≥75 分可发布；60-74 分需优化；<60 分废弃重做 |

## 四、叙事型软广视频评分总览（生成后评估）

叙事型最终得分 = **直给型五维分（100 分制） + 叙事专项分（45 分制）**，满分 **145 分**。

**建议发布阈值**：
- ≥ 110 分：可发布
- 90-109 分：微调后发布
- < 90 分：重新构思

### 4.1 叙事专项评分细则（45 分）

| 指标 | 分值 | 评估标准 |
| :--- | :--- | :--- |
| **悬念维持度** | 10 分 | 前 12 秒是否建立困境？12-22 秒悬念是否持续增强？观众是否有理由看到最后？ |
| **情绪弧线完整度** | 10 分 | 是否有清晰的情绪变化轨迹（疲惫→好奇→惊讶→治愈）？情绪转折是否自然？ |
| **产品植入自然度** | 10 分 | 产品是否作为“故事道具”出现而非“广告主角”？是否无硬广痕迹？ |
| **评论区引导力** | 10 分 | 视频内容是否自然激发“WHAT IS THIS?”类评论？悬念是否驱动用户留言？ |
| **零提及纪律** | 5 分 | 视频内是否严格避免品牌名/Logo/价格/购买引导语？ |


## 五、失败模式及修复方案速查表

| 失败表现 | 可能原因 | 修复方案 |
| :--- | :--- | :--- |
| **三层诊断致命层不通过** | 钩子无视觉冲突/无声音钩子 | 退回阶段 2，强化钩子和声音策略 |
| **三层诊断核心层不通过（垂直信号缺失，扣5分）** | 前5秒未口述+字幕双重强化核心主题词 | 在前5秒增加口述和字幕的关键词强化 |
| **三层诊断核心层不通过（原生感未启用，扣3分）** | 提示词中缺少原生感关键词 | 添加 `Natural window light`, `Handheld POV`, `Authentic unscripted feel` 等词汇 |
| **三层诊断卫生层不通过** | 含否定式负向词/超过100词 | 自动替换为正向锚点词，拆分过长段落 |
| 前 3 秒平淡 | 钩子类型选错或未强调开场 | 对照品类选型表换钩子；前3秒增加 ASMR/音效 |
| **播放量卡在 200-500** | 未通过测试池，前3秒留存不足 | 强化前3秒声音钩子+视觉冲击；美国市场切叙事型 |
| 完播率低，中间划走 | 信息密度不足，有空镜 | 增加指令：`Snappy motion, Quick cuts` |
| 收藏率低 | 未强调实用价值或下次使用 | 增加引导语 |
| 分享率低 | 未赋予社交货币 | 对照品类更换社交货币类型 |
| YouTube Shorts 推荐量低 | 前 2 秒划走率高 | 前2秒声音钩子+极端特写 |
| Meta 限流 | 内容过于病毒式煽动 | 启用原生感策略 |
| AIGC 标签未勾选 | 发布时未提醒合规操作 | 发布前强制提醒 |
| **SEO 搜索流量低** | 提示词未植入关键词视觉元素 | 添加 `Search-friendly visual`、前5秒口述+字幕核心关键词 |
| **指令超长导致画面异常** | 提示词超过 200 词 | 改分段生成，每段≤100 词 |
