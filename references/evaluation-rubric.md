# 视频质量与爆款潜力评分表 (Evaluation Rubric) v2.6

> **用途**：对 Seedance 2.0 生成的视频进行标准化评估，判断是否达到发布标准，并为迭代提供数据依据。
> **评分原则**：满分 100 分，分为“技术质量”“爆款钩子”“平台适配”“导演执行”“算法信号”五大维度。
> **更新版本**：v2.6 (2026.05) —— 统一版本号，新增“原生感执行度”子项，微调评分细则对齐 TikTok 小批量测试池（200-500人）与 YouTube Shorts 划走率指标。

---

## 一、评分总览

| 维度 | 分值 | 评估重点 |
| :--- | :--- | :--- |
| **技术质量** | 20 分 | 画面清晰度、运镜质感、AI 生成瑕疵 |
| **爆款钩子** | 30 分 | 前 3 秒冲击力、完播率潜力、复播引导设计 |
| **平台适配** | 20 分 | 收藏/分享引导力、TikTok SEO 潜力、Meta 真实价值感、静音可读性 |
| **导演执行** | 15 分 | 五维架构执行度、原生感执行度、多镜头结构完整性、音频同步效果、转场质量 |
| **算法信号** | 15 分 | 复播率预估、收藏引导力预估、分享引导力预估 |
| **总分** | **100 分** | ≥75 分可发布；60-74 分需优化；<60 分废弃重做 |

## 二、技术质量评分细则 (20 分)

### 2.1 画面清晰度与分辨率 (6 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 6 分 | 画面锐利，1080x1920 满分辨率，无模糊、马赛克、噪点 |
| 4-5 分 | 轻微柔化但主体清晰，分辨率达标 |
| 2-3 分 | 局部模糊，边缘有锯齿，影响观感 |
| 0-1 分 | 大面积模糊、变形、画质低劣，不可用 |

### 2.2 运镜质感与构图 (6 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 6 分 | 运镜丝滑自然，镜头运动（推拉摇移跟）执行精准，主体始终处于视觉中心，构图符合 9:16 竖屏美学 |
| 4-5 分 | 轻微抖动但不影响观看，运镜意图可辨但不够精准 |
| 2-3 分 | 晃动明显或构图失衡，运镜与内容不匹配 |
| 0-1 分 | 剧烈抖动、眩晕感、主体出画，或完全无运镜设计 |

### 2.3 AI 生成瑕疵控制 (8 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 8 分 | 无人物手脸变形、物体形变、鬼影、闪烁等 AI 常见瑕疵 |
| 5-7 分 | 轻微瑕疵但不被普通观众察觉（如指尖略糊） |
| 3-4 分 | 明显瑕疵引起注意（如手指数量异常、物体融化解体） |
| 0-2 分 | 严重畸变、恐怖谷效应，直接劝退观众 |

## 三、爆款钩子评分细则 (30 分)

### 3.1 前 3 秒留存预测 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 首帧即含“认知失调”或“视觉奇观”，让人本能停止划动 |
| 6-8 分 | 有明显视觉冲击或悬念，但不够极致 |
| 3-5 分 | 平淡开场，需要观众耐心等待信息 |
| 0-2 分 | 前 3 秒完全是空镜、黑屏、无关画面，必划走 |

### 3.2 完播率潜力 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 15 秒内信息密度高，无冗余帧，每 3 秒有“小钩子”，结尾有定格引导 |
| 6-8 分 | 整体流畅但有 1-2 秒拖沓，结尾无明显停留 |
| 3-5 分 | 节奏松散，中间有无效空镜，结尾仓促 |
| 0-2 分 | 大量无效时长，观众无法坚持看完 |

### 3.3 复播引导设计 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 内置明确复播彩蛋（信息差、节奏中断、双重结局），观众有明显回看动机 |
| 6-8 分 | 有一定复播引导但不够突出或自然 |
| 3-5 分 | 无意间可能触发复播，但无主动设计 |
| 0-2 分 | 无任何复播引导，一次看完即走 |

## 四、平台适配评分细则 (20 分)

### 4.1 收藏/分享引导力 (8 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 7-8 分 | 明确引导**收藏**和**社交货币分享**，理由充分（实用价值锚定、展现品味、圈层归属），话术与视觉配合到位 |
| 5-6 分 | 有引导但理由不够充分或只有其中一项 |
| 3-4 分 | 仅有浅层引导（如“点赞关注”），未涉及收藏/分享 |
| 0-2 分 | 无任何收藏/分享引导 |

| 2-3 分 | 有一定讨论价值，但引导不明确或偏浅层 |
| 1 分 | 内容看完即走，无评论/分享理由 |
| 0 分 | 内容封闭，无互动可能 |

| 2 分 | 内容可被搜索但关键词不够突出，或前 2 秒有划走风险 |

## 五、导演执行评分细则 (15 分)

### 5.1 五维架构执行度 (4 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 4 分 | 五维（技术基底/镜头运动/视觉元素/光影系统/动态设计）均明确且执行到位，画面具有电影感 |
| 2-3 分 | 缺少 1-2 个维度或执行不够精准，整体效果尚可 |
| 1 分 | 仅依赖描述性语言，未使用结构化指令，画面平淡 |
| 0 分 | 完全无结构，生成效果随机 |

### 5.2 原生感执行度 (3 分) —— 新增

| 得分 | 标准描述 |
| :--- | :--- |
| 3 分 | 视频风格符合 UGC 特征：素人演员感、生活化场景、自然光线、手持自拍感、轻微瑕疵可见，无明显广告精致感 |
| 2 分 | 部分符合原生感特征，但仍有明显摆拍或影棚痕迹 |
| 1 分 | 高度精修广告风格，与原生感策略相悖（除非产品需要） |
| 0 分 | 完全未考虑原生感 |

### 5.3 多镜头结构完整性 (3 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 3 分 | 镜头数量与品类模板匹配（3-4 镜头），各镜头时长分配合理，转场自然流畅 |
| 2 分 | 有明显镜头切换但节奏或转场略生硬，或镜头数量不足 |
| 1 分 | 尝试多镜头但切换混乱，破坏叙事 |
| 0 分 | 单镜头无切换，或切换完全失败 |

### 5.4 音频同步效果 (2 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 2 分 | 画面运动与音频节奏/重音明显同步，视听体验增强 |
| 1 分 | 有同步意图但不够精准，或音频与画面割裂 |
| 0 分 | 无音频或完全无关 |

### 5.5 转场质量 (3 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 3 分 | 转场方式与内容高度契合，无 AI 形变，执行精准 |
| 2 分 | 转场基本流畅但有轻微瑕疵 |
| 1 分 | 转场处出现明显 AI 变形或跳帧 |
| 0 分 | 无转场或转场完全失败 |

## 八、失败模式及修复方案速查表 (v2.6 扩充)

| 失败表现 | 可能原因 | 修复方案 |
| :--- | :--- | :--- |
| 前 3 秒平淡 | 钩子类型选择错误或未强调开场 | 强制使用爆款前缀：`High CTR style, Eye-catching opening` |
| 完播率低，中间划走 | 信息密度不足，有空镜 | 增加指令：`Snappy motion, No empty frames` |
| **新视频播放量卡在 200-500** | 未通过 TikTok 小批量测试池 | 强化前 3 秒冲击力，增加收藏/分享引导，确保首帧即钩子 |
| **YouTube Shorts 推荐量低** | 前 2 秒划走率高 | 前 2 秒必须制造新奇反应，避免缓慢空镜 |
| **运镜质感差，画面呆板** | 未使用标准化运镜指令 | 从运镜词库选择：`Slow dolly-in` / `Slight orbit` / `Whip pan` |
| **五维架构缺失，画面无电影感** | 仅用描述性语言，未结构化 | 补充技术基底、光影系统、动态设计维度 |
| **原生感不足，广告味太重** | 未启用原生感策略 | 添加：`Authentic unscripted reaction, Lived-in messy background, Natural window light` |
| **复播率低** | 缺乏回看动机 | 添加复播彩蛋：`Brief flash of surprising detail at 3s` + 文字“Watch again” |
| **收藏率低** | 未强调实用价值或下次使用 | 增加引导语：“Save for your next ___” + 收藏图标动画 |
| **分享率低** | 未赋予社交货币 | 增加社交货币话术：“Share this if you have good taste” / “Tag your ___ bestie” |
| 人物面部/手部崩坏 | Seedance 对人物细节处理不稳定 | 添加：`Well-formed hands, Natural facial features, No distortion` |
| 画面抖动眩晕 | “手持真实感”过度 | 改为：`Smooth camera motion, Subtle handheld breathing` |
| 静音看不懂 | 未添加文字叠加层 | 加入：`Clear space for bold text overlay, Self-explanatory visual` |
| 多镜头切换处变形 | 转场指令不够明确 | 使用精确转场词：`Quick whip pan transition` 或 `Match cut on hand movement` |
| 音频与画面不同步 | 未添加音频同步指令 | 添加：`Visual beats sync with heavy bass drops` |
| Meta 限流（真实兴趣不足） | 内容过于病毒式煽动 | 减少夸张前缀，增加 `Authentic real-life moment, True interest content` |
