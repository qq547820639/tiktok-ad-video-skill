# 视频质量与爆款潜力评分表 (Evaluation Rubric) v2.2

> **用途**：对 Seedance 2.0 生成的视频进行标准化评估，判断是否达到发布标准，并为迭代提供数据依据。
> **评分原则**：满分 100 分，分为“技术质量”“爆款钩子”“平台适配”“进阶叙事”“算法信号”五大维度。
> **更新版本**：v2.2 (2026.04) —— 拆分完播率与复播率，细化互动质量评估，新增 Meta 真实兴趣信号预估。

---

## 一、评分总览

| 维度 | 分值 | 评估重点 |
| :--- | :--- | :--- |
| **技术质量** | 20 分 | 画面清晰度、运镜稳定性、AI 生成瑕疵 |
| **爆款钩子** | 30 分 | 前 3 秒冲击力、完播率潜力、复播引导设计 |
| **平台适配** | 20 分 | TikTok 评论/分享引导、YouTube SEO 潜力、Meta 真实价值感、静音可读性 |
| **进阶叙事** | 15 分 | 多镜头结构完整性、音频同步效果、转场质量 |
| **算法信号** | 15 分 | 复播率预估、互动质量预估、收藏/分享引导力 |
| **总分** | **100 分** | ≥75 分可发布；60-74 分需优化；<60 分废弃重做 |

## 二、技术质量评分细则 (20 分)

### 2.1 画面清晰度与分辨率 (6 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 6 分 | 画面锐利，1080x1920 满分辨率，无模糊、马赛克、噪点 |
| 4-5 分 | 轻微柔化但主体清晰，分辨率达标 |
| 2-3 分 | 局部模糊，边缘有锯齿，影响观感 |
| 0-1 分 | 大面积模糊、变形、画质低劣，不可用 |

### 2.2 运镜流畅度与构图 (6 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 6 分 | 运镜丝滑自然，主体始终处于视觉中心，构图符合 9:16 竖屏美学 |
| 4-5 分 | 轻微抖动但不影响观看，主体位置基本合理 |
| 2-3 分 | 晃动明显或构图失衡，主体被裁切 |
| 0-1 分 | 剧烈抖动、眩晕感、主体出画 |

### 2.3 AI 生成瑕疵控制 (8 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 8 分 | 无人物手脸变形、物体形变、鬼影、闪烁等 AI 常见瑕疵 |
| 5-7 分 | 轻微瑕疵但不被普通观众察觉（如指尖略糊） |
| 3-4 分 | 明显瑕疵引起注意（如手指数量异常、物体融化解体） |
| 0-2 分 | 严重畸变、恐怖谷效应，直接劝退观众 |

## 三、爆款钩子评分细则 (30 分)

### 3.1 前 3 秒留存预测 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 首帧即含“认知失调”或“视觉奇观”，让人本能停止划动 |
| 6-8 分 | 有明显视觉冲击或悬念，但不够极致 |
| 3-5 分 | 平淡开场，需要观众耐心等待信息 |
| 0-2 分 | 前 3 秒完全是空镜、黑屏、无关画面，必划走 |

### 3.2 完播率潜力 (10 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 15 秒内信息密度高，无冗余帧，每 3 秒有“小钩子”，结尾有定格引导 |
| 6-8 分 | 整体流畅但有 1-2 秒拖沓，结尾无明显停留 |
| 3-5 分 | 节奏松散，中间有无效空镜，结尾仓促 |
| 0-2 分 | 大量无效时长，观众无法坚持看完 |

### 3.3 复播引导设计 (10 分) —— 新增

| 得分 | 标准描述 |
| :--- | :--- |
| 9-10 分 | 内置明确复播彩蛋（信息差、节奏中断、双重结局），观众有明显回看动机 |
| 6-8 分 | 有一定复播引导但不够突出或自然 |
| 3-5 分 | 无意间可能触发复播，但无主动设计 |
| 0-2 分 | 无任何复播引导，一次看完即走 |

## 四、平台适配评分细则 (20 分)

### 4.1 TikTok 适配 (6 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 5-6 分 | 内容具备**评论**或**分享**动机，结尾有明确引导，且引导语鼓励深度评论（非 emoji） |
| 3-4 分 | 有一定讨论价值，但引导不明确或偏浅层 |
| 1-2 分 | 内容看完即走，无评论/分享理由 |
| 0 分 | 内容封闭，无互动可能 |

### 4.2 YouTube Shorts 适配 (5 分)

| 得分 | 标准描述 |
| :--- | :--- |

| 2-3 分 | 内容可被搜索但关键词不够突出 |
| 1 分 | 内容泛化，难以匹配具体搜索意图 |
| 0 分 | 纯氛围内容，无搜索价值 |

### 4.3 Meta Reels 适配 (5 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 4-5 分 | 内容传递**真实价值**，前 5 秒亮明品牌/产品，符合 Meta UTIS 模型（音频质量、制作风格、情感基调一致） |
| 2-3 分 | 有价值输出但品牌露出不足或风格略浮夸 |
| 1 分 | 偏病毒式娱乐，真实价值感弱 |
| 0 分 | 空洞口号、过度夸张、无实质信息 |

### 4.4 Pinterest 适配 (4 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 4 分 | 静音状态下完全可懂，画面有大号文字叠加，封面帧具备高点击率 |
| 2-3 分 | 静音可懂但文字偏小或不突出 |
| 1 分 | 依赖音频理解内容 |
| 0 分 | 无音频则完全不知所云 |

## 五、进阶叙事评分细则 (15 分)

### 5.1 多镜头结构完整性 (5 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 5 分 | 双镜头结构清晰（如 0-7s 痛点 + 8-15s 解决），转场自然流畅，叙事完整 |
| 3-4 分 | 有明显镜头切换但节奏或转场略生硬 |
| 1-2 分 | 尝试多镜头但切换混乱，破坏叙事 |
| 0 分 | 单镜头无切换，或切换完全失败 |

| 得分 | 标准描述 |
| :--- | :--- |

| 3-4 分 | 有同步意图但不够精准 |
| 1-2 分 | 音频与画面割裂 |
| 0 分 | 无音频或完全无关 |

### 5.3 双镜头转场质量 (5 分)

| 得分 | 标准描述 |
| :--- | :--- |
| 5 分 | 转场方式（Whip pan / Match cut / 手部遮挡）与内容高度契合，无 AI 形变 |
| 3-4 分 | 转场基本流畅但有轻微瑕疵 |
| 1-2 分 | 转场处出现明显 AI 变形或跳帧 |
| 0 分 | 无转场或转场完全失败 |

## 八、失败模式及修复方案速查表 (v2.2 扩充)

| 失败表现 | 可能原因 | 提示词/策略修复方案 |
| :--- | :--- | :--- |
| 前 3 秒平淡 | 钩子类型选择错误或未强调开场 | 强制使用爆款前缀：`High CTR style, Eye-catching opening` |
| 完播率低，中间划走 | 信息密度不足，有空镜 | 增加指令：`Snappy motion, No empty frames` |
| **复播率低** | 缺乏回看动机 | 添加复播彩蛋：`Brief flash of surprising detail at 3s` + 文字“Watch again” |
| **互动质量差（全是 emoji）** | 互动引导过于浅层 | 改用开放式提问或争议观点引导深度评论 |
| **收藏/分享少** | 未强调实用价值或社交货币 | 增加引导语：“下次...前记得回来看” / “转给需要的人” |
| 人物面部/手部崩坏 | Seedance 对人物细节处理不稳定 | 添加：`Well-formed hands, Natural facial features, No distortion` |
| 画面抖动眩晕 | “手持真实感”过度 | 降权手持词，改为：`Smooth camera motion, Subtle handheld breathing` |
| 静音看不懂 | 未添加文字叠加层 | 加入：`Clear space for bold text overlay, Self-explanatory visual` |
| TikTok 评论少 | 内容缺乏讨论点 | 结尾增加开放式提问或争议观点 |
| YouTube 无搜索流量 | 标题/描述未做 SEO | 发布阶段补关键词，画面中加入关键词视觉元素 |
| **Meta 限流（真实兴趣不足）** | 内容过于病毒式煽动，风格浮夸 | 减少夸张前缀，增加 `Authentic real-life moment, True interest content`，确保 UTIS 一致性 |
| 多镜头切换处变形 | 转场指令不够明确 | 使用精确转场词：`Quick whip pan transition` 或 `Match cut on hand movement` |
| 音频与画面不同步 | 未添加音频同步指令 | 添加：`Visual beats sync with heavy bass drops` |
