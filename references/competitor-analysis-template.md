# 竞品爆款拆解模板 (Competitor Analysis Template) v1.0

> **用途**：为 `SKILL.md` 阶段 -1「竞品爆款拆解」提供结构化的拆解框架与记录模板，帮助用户系统分析同类目 Top 爆款视频，提取可复用的结构要素。
> **使用方式**：针对每个竞品视频填写一张拆解表，完成后进行共性提取，输出该类目爆款结构共性报告。
> **创建版本**：v1.0 (2026.05) —— 初始版本。

## 四、拆解示例（铸铁锅类目）

### 示例视频拆解

| 字段 | 内容 |
| :--- | :--- |
| **视频 ID / 链接** | TikTok @castironmaster / 视频 ID: 7xxxxx |
| **产品品类** | 锅具/厨房用品 |
| **发布账号粉丝量** | 85K（垂类达人） |
| **播放量 / 互动数据** | 4.2M 播放，收藏率 10%，分享率 7% |
| **视频时长** | 15s（直给型） |
| **内容形态** | 直给型 |

#### 1. 钩子拆解

| 子项 | 内容 |
| :--- | :--- |
| **钩子类型** | 视觉奇观型 |
| **前 3 秒具体画面** | 干锅打入一颗鸡蛋，蛋液顺滑滑动如溜冰 |
| **前 3 秒声音策略** | 纯 ASMR 煎蛋滋滋声，无口播 |
| **口播进入时机** | 第 3 秒进入 |
| **钩子核心文案** | 字幕："ZERO OIL. 100% SLIDE." |

#### 2. 脚本结构拆解

| 镜头 | 时长 | 画面描述 | 字幕/大字报 | 声音策略 |
| :--- | :--- | :--- | :--- | :--- |
| 镜头 1 | 0-3s | 干锅打蛋，蛋液滑动 | "ZERO OIL. 100% SLIDE." | ASMR 滋滋声 |
| 镜头 2 | 3-7s | 微距特写蛋液边缘微焦，铲子轻推整体滑动 | "NO STICK. EVER." | ASMR 滑动声 |
| 镜头 3 | 7-12s | 慢动作蛋饼翻面，表面金黄均匀 | "GOLDEN EVERY TIME" | 画面随翻面声同步 |
| 镜头 4 | 12-15s | 定格完美煎蛋，铲轻敲锅边 | "Save this pan hack 📌" | 收银机叮声 |

**镜头总数**：4
**转场方式**：Whip pan

#### 3. 互动策略拆解

| 子项 | 内容 |
| :--- | :--- |
| **收藏引导话术** | "Save this pan hack 📌"（画面文字） |
| **分享引导话术** | 置顶评论："Tag a friend who needs a better pan 🍳" |
| **评论区置顶/互动** | 主号置顶："What's your biggest pan struggle?" |
| **垂直领域信号** | 前 5 秒字幕："uncoated cast iron" |

#### 4. 视觉风格拆解

| 子项 | 内容 |
| :--- | :--- |
| **光影风格** | 自然窗光 + 暖色调温馨感 |
| **运镜特征** | 微距特写为主，手持轻微呼吸感 |
| **原生感程度** | 高（素人手部，真实厨房背景） |
| **人物特征** | 25-35 岁女性素人手部出镜 |


## 五、快速分析检查清单

分析竞品视频时，重点关注以下问题：

- [ ] 前 3 秒有没有让我“停住划动”的冲动？
- [ ] 声音是什么时候进入的？口播第几秒？
- [ ] 视频看完后，我最想点哪个互动按钮？为什么？
- [ ] 这个产品的核心卖点，视频是用画面证明还是用语言说明？（前者更强）
- [ ] 如果去掉所有文字和口播，我还能看懂产品好不好吗？
