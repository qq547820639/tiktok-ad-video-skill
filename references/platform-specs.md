## 二、平台详细规格与算法深度解析

### 🎬 TikTok

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16)，最低 540x960 |
| 帧率 | 24/25/30fps (推荐 24fps) |
| 编码格式 | H.264 或 H.265 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3 Mbps (推荐 6-8 Mbps) |
| 色彩空间 | Rec. 709 |

#### AIGC 合规要求
- TikTok 要求对 AI 生成内容进行标签披露。发布时需勾选 **“AI-generated content”** 选项。
- 未标注可能导致内容被下架或限流。

#### 2026 算法核心变化
1. **小批量测试分发**：新视频发布后，首先进入约 **200-500 人的小批量测试池**（兴趣匹配受众）。测试池的反馈（完播率、收藏、分享）决定能否进入下一级流量池。
2. **垂直领域强化**：2026 年算法会自动识别视频主题标签，并优先推送给对该领域有持续观看行为的用户。**前 5 秒内必须通过口述+字幕双重强化核心主题词**。
3. **算法更新频率提升至月度**。
4. **互动权重重塑**：收藏(Save)和分享(Share)权重首次超过点赞。
5. **TikTok 搜索成为核心信号**：标题、字幕中的关键词成为核心流量入口。
6. **完播率与重播率**：完播率门槛提升至70%，重播率超20%被视为优秀水平。

#### TikTok SEO 策略
| 优化点 | 具体做法 |
| :--- | :--- |
| **语音关键词** | 视频前 5 秒口述核心主题词（权重最高） |
| **字幕关键词** | 视频内嵌字幕包含产品名称和核心卖点词，可被搜索索引（权重次之） |
| **标题关键词** | 包含 1-2 个核心搜索关键词，放在标题前部（权重较弱） |
| **话题标签** | 3-5 个垂直标签，其中 1 个为长尾关键词标签 |
| **评论区置顶** | 置顶评论中包含核心关键词，增强搜索相关性 |

#### 发布优化清单
- 标题含 1-2 个核心关键词 + 1 个热门标签。
- **前 5 秒口述+字幕双重强化核心主题词**，帮助算法精准识别垂直领域。
- 评论区置顶引导互动，设计深度评论问题。
- 优先引导**收藏(Save)** 和**社交货币分享(Share)**。
- 背景音乐选择 Trending 榜单前 20，确保画面与节拍同步。
- **密切关注月度算法更新**，及时调整策略。
- **发布时务必勾选“AI-generated content”标签**。


### ▶️ YouTube Shorts

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16) |
| 帧率 | 24/30/60fps |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3.5 Mbps |
| 音频 | AAC 或 MP3，≥128kbps |

#### AIGC 合规要求
- YouTube 要求对经过 AI 修改或生成的内容进行披露。发布时需勾选 **“Altered content”** 选项。

#### 2026 算法核心变化
1. **划走率 (Swipe Away Rate) 是关键负向指标**：前 2 秒必须制造新奇反应。
2. **多平台分享加权**：将 Shorts 链接分享至 X/Twitter、Facebook 等社群平台，可获得额外推荐加权。
3. **搜索成为关键流量来源**：2026年1月新增 Shorts 专属搜索过滤器。
4. **重播率是推荐核心**：完成率达到70%以上的 Shorts 会获得激进推广。

#### 发布优化清单
- **前 2 秒必须抓人**，避免任何缓慢空镜。
- 标题必须做 SEO：核心关键词前置，约40字符内。
- 描述区前 100 字符包含核心关键词，并添加 3-5 个相关标签。
- **引导观众将视频分享至 X/Twitter 或 Facebook**，获取多平台分享加权。
- **发布时务必勾选“Altered content”披露选项**。

## 三、AIGC 标签流量影响观察与对照实验设计

> **背景**：2026 年各平台已全面落地 AIGC 内容标签强制披露政策。根据创作者社区反馈，勾选标签后可能存在隐性流量影响。

### 3.1 各平台 AIGC 标签对自然流量的潜在影响

| 平台 | 标签名称 | 是否强制 | 流量影响观察（基于社区反馈） |
| :--- | :--- | :--- | :--- |
| **TikTok** | AI-generated content | ✅ 强制 | 部分创作者反馈勾选后自然推荐量下降 10-20%，平台未公开承认 |
| **Meta (IG/FB)** | Made with AI | ✅ 强制 | 审核较严，未勾选可能导致拒审；勾选后流量影响暂无明显规律 |
| **YouTube** | Altered content | ✅ 强制 | 对 Shorts 推荐影响较小，长视频影响略大（约 5-15%） |
| **Pinterest** | 暂无强制标签 | ⚠️ 建议主动标注 | 未标注无直接影响，但主动标注可增强品牌透明度 |
| **Snapchat** | 暂无强制标签 | ⚠️ 建议主动标注 | 未强制，但平台正在测试 AIGC 识别能力 |

### 3.2 对照实验设计建议

**实验步骤**：
1. **准备两条内容相同、钩子相同的视频**。
2. **视频 A**：发布时勾选平台的 AIGC 标签。
3. **视频 B**：发布时不勾选 AIGC 标签（需确认平台允许）。
4. **同时发布**：两条视频在相近时段（24h 内）发布到同一账号。
5. **观察 48 小时数据**：对比两者的播放量、完播率、收藏率、分享率。

### 3.3 最小化 AI 感的策略（规避强制标注）

| 策略 | 具体做法 | 适用场景 |
| :--- | :--- | :--- |
| **启用原生感策略** | 使用素人演员、生活化场景、自然光线、手持自拍感词汇 | 所有品类 |
| **混剪实拍素材** | 视频中穿插 2-3 秒真实产品实拍镜头 | 有产品实拍条件的用户 |
| **减少精修痕迹** | 避免过度完美的光影和运镜，保留轻微瑕疵 | 追求真实感的品类 |
| **降低运镜复杂度** | 使用简单的固定镜头或轻微手持晃动 | Fast 模式或规避 AI 感时 |


## 四、美国 TikTok 评论区运营规则（v2.10 新增）

> **适用场景**：美国市场叙事型软广发布后的评论区运营。直给型广告也可参考，但叙事型对评论区的依赖度更高。

### 4.1 为什么评论区运营比视频内容更重要

2026 年 TikTok 算法权重分配（美国市场估算）：
- 完播率：30%
- **评论量 + 评论互动率：25%**
- 分享率：20%
- 点赞率：15%
- 收藏率：10%

评论区活跃度直接影响是否突破测试池。叙事型软广的转化路径是 **视频内零提及 → 评论区自然追问 → 主号回复植入产品信息**，因此评论区运营是转化漏斗的核心环节。

### 4.2 评论区运营时间线

| 时间节点 | 动作 | 目的 |
| :--- | :--- | :--- |
| 发布后 0-3 分钟 | 用 2-3 个小号发布“好奇心”评论 | 制造初始互动，骗过冷启动算法 |
| 发布后 3-10 分钟 | 主号回复小号，自然植入产品信息 | 引导真实用户追问产品 |
| 发布后 10-60 分钟 | 回复每一条真实评论 | 提升评论互动率，触发算法推流 |
| 发布后 1-6 小时 | 用评论回复引导“第二条视频”话题 | 为续集内容埋钩子 |
| 发布后 6-48 小时 | 持续回复，每 2 小时检查一次 | 维持评论区热度 |

### 4.3 小号评论模板库

**好奇心型**（驱动用户追问产品）：
- "wait WHAT is that?? I need answers 😭"
- "ok but what IS that thing"
- "I've watched this 5 times and I still don't understand what that is"

**效果惊叹型**（强化产品效果认知）：
- "the way it just SLID off the pan... excuse me??"
- "no because HOW"
- "this is witchcraft and I'm here for it"

**社交分享型**（驱动分享行为）：
- "sending this to everyone I know"
- "tagging my roommate because we NEED this"
- "this is the content I come to tiktok for"

**购买意向型**（最有价值，直接驱动转化）：
- "shut up and take my money"
- "adding to cart immediately"
- "where do I get one?? asking for myself and everyone I know"

### 4.4 主号回复话术库

| 用户问 | 主号回复 |
| :--- | :--- |
| "what is this?" | "it's a cast iron pan! the shape is called yuanbao, it's a traditional Chinese design — link in my bio" |
| "where did you get it?" | "I put the link in my bio! fair warning shipping took like 2 weeks" |
| "how much?" | "I think it was around $[价格] — not bad for cast iron honestly" |
| "does it really not stick?" | "only if you season it right! I can make a video on that if you want 👀" |
| "is it heavy?" | "honestly yeah it's heavier than a regular pan but you get used to it. the weight is what makes it cook so evenly" |

### 4.5 铁律

1. **永远不要复制粘贴相同回复**（TikTok 会检测并降权）。
2. **每条回复都像朋友聊天**（不用“您好”、不用客服语气）。
3. **用回复引导更多互动**（"I can make a video on that if you want" → 为第二条视频埋钩子）。
4. **不要在视频内放任何购买引导**（全部后置到评论区和 bio）。
5. **小号之间不要互相回复**（看起来像水军会被标记）。


## 五、跨平台通投策略

### 5.1 统一素材标准
- 直给型：母版视频统一按 **9:16 (1080x1920)**、**15 秒**、**品类匹配多镜头模板（3-4 镜头）** 制作。
- 叙事型：分段生成 4 段 15 秒片段后拼接为 45-60 秒视频。
- **前 5 秒口述+字幕双重强化核心主题词**，确保 TikTok 垂直领域信号和 SEO 优化同步完成。

### 5.2 平台差异化发布清单

| 发布要素 | TikTok | YouTube Shorts | Reels (IG/FB) | Pinterest |
| :--- | :--- | :--- | :--- | :--- |
| **标题风格** | 口语化 + 引导评论/收藏 | SEO 优化 + 悬念词 | 价值陈述 + 信任建立 | 关键词堆叠 + 灵感描述 |
| **SEO 重点** | 语音关键词(前5秒) > 字幕 > 标题 | 标题+描述关键词 | 字幕关键词 | 标题+描述关键词 |
| **标签数量** | 3-5 个 | 3-5 个长尾 | 2-3 个垂直 | 5-10 个关键词 |
| **音乐** | 必选热门音 | 可选 | 推荐版权库 | 不重要 |
| **文字叠加** | 适量 | 适量 | 推荐原生工具 | 必须加粗大字 |
| **发布频率** | 每日 1-3 条 | 每日 1-2 条 | **每日 1-3 条（分散发布）** | 每周 5-10 条 |
| **特别关注** | **垂直领域信号+突破测试池** | **降低前2秒划走率** | **当日发布优先** | **多格式发布加权** |
| **AIGC标签** | AI-generated content | Altered content | Made with AI | 建议主动标注 |

### 5.3 避免跨平台禁忌
- **水印问题**：任何含 TikTok 水印的视频不要直接上传至 Reels 或 Spotlight，会被严重限流。
- **音乐版权**：TikTok 热门音乐在 Reels 上可能无版权，发布前检查 Instagram 音乐库。
- **转场兼容**：快速摇摄 (Whip pan) 在所有平台均兼容，是首选转场方式。
