## 二、平台详细规格与算法深度解析

### 🎬 TikTok

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16)，最低 540x960 |
| 帧率 | 24/25/30fps (推荐 24fps) |
| 编码格式 | H.264 或 H.265 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3 Mbps (推荐 6-8 Mbps) |
| 色彩空间 | Rec. 709 |

#### 2026 算法核心变化
1. **粉丝优先测试**：内容发布后优先推给粉丝，他们的反馈决定能否进入公域流量池。
2. **互动权重重塑**：收藏(Save)和分享(Share)权重首次超过点赞，是最重要的深度互动信号。
3. **TikTok搜索**：正式成为核心推荐信号，标题、字幕中的关键词成为流量入口。
4. **完播率与重播率**：完播率门槛提升至70%，重播率超20%被视为优秀水平。
5. **二次传播**：推流机制转为“完播率+二次传播”双核心模型，没有保存和分享的完播价值将大打折扣。

#### 发布优化清单
- 标题含 1-2 个核心关键词 + 1 个热门标签。
- 评论区置顶引导互动，设计深度评论问题。
- 优先引导**收藏(Save)** 和**分享(Share)**，这是2026年最高权重信号。
- 背景音乐选择 Trending 榜单前 20，确保画面与节拍同步。
- 双镜头视频在文案中暗示“看到最后”，提升完播期待。

### ▶️ YouTube Shorts

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16) |
| 帧率 | 24/30/60fps |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3.5 Mbps |
| 音频 | AAC 或 MP3，≥128kbps |

#### 2026 算法核心变化
1. **搜索成为关键流量来源**：2026年1月，YouTube 新增 Shorts 专属搜索过滤器，用户可专门筛选 Shorts 内容。
2. **重播率是推荐核心**：完成率达到70%以上的 Shorts 会获得激进推广。
3. **评论和分享权重高于点赞**：对分发影响更大。
4. **长视频联动**：Shorts 可关联长视频，相互反哺。

#### 发布优化清单
- 标题必须做 SEO：核心关键词前置，约40字符内。
- 描述区前 100 字符包含核心关键词，并添加 3-5 个相关标签。
- 首镜头应包含可被搜索索引的关键词视觉元素。
- 视频末尾 2 秒添加“订阅”动画按钮。

### 📷 Instagram Reels

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16) |
| 帧率 | 30fps 最低 |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3.5 Mbps |
| 音频 | 推荐使用 Instagram 音乐库内版权音乐 |

#### 2026 算法核心变化
1. **“真实兴趣” (UTIS) 模型**：Meta 引入用户真实兴趣调查，不再只依赖点赞、观看时长等行为信号。
2. **有效完播密度**：用户在播放过程中无分心行为的完播才计为有效。
3. **搜索优先 (Instagram SEO)**：搜索排名受账号名称、Bio、Caption、Reels 字幕影响，Hashtag 影响力下降。
4. **内容新鲜度**：算法优先推荐当天发布的内容，50% 以上的 Reels 推荐来自当日发布。

#### 发布优化清单
- 前 3 秒亮明品牌/产品名，建立信任预期。
- 双镜头结构：镜头一展示问题，镜头二展示产品解决方案。
- 使用 Instagram 内置文字工具添加字幕，提升搜索可见性。
- 避免过度夸张的病毒式文案，强调真实价值。

### 👥 Facebook Reels

#### 技术规格
与 Instagram Reels 完全一致。

#### 2026 算法核心变化
1. **熟人社交优先**：优先分发给用户的好友网络。
2. **价值输出型内容受宠**：教程、科普、生活技巧类视频获得更多推荐。
3. **品牌透明度要求**：前 5 秒内必须明确呈现品牌/产品信息。

#### 发布优化清单
- 前 5 秒以字幕或特写形式清晰展示品牌/产品名。
- 双镜头结构：镜头一快速建立问题场景，镜头二展示解决方案并突出品牌。
- 引导用户标记朋友（如“Tag a friend who needs this”）。

### 👻 Snapchat Spotlight

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16)，最低 720x1280 |
| 帧率 | 30fps |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3 Mbps |
| 音频 | 立体声，≥128kbps |
| 水印 | 禁止包含任何水印 |

#### 2026 算法核心
1. **完播率是唯一王者**：完播率是最强的预测指标。
2. **年轻化审美**：核心用户 13-24 岁，偏好快节奏、无废话内容。
3. **无公开评论区**：互动信号主要依赖重播、点赞和分享。

#### 发布优化清单
- 视频前 1 秒内必须进入核心动作，零铺垫。
- 时长控制在 15-25 秒。
- 去除所有水印后再上传。
- 双镜头转场使用快速摇摄 (Whip pan)，不要拖沓。

### 📌 Pinterest

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16)，也支持 1080x1080 (1:1) |
| 帧率 | 25/30fps |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3 Mbps |
| 音频 | **不重要**，默认静音播放 |

#### 2026 算法核心
1. **视觉自解释力**：90% 用户在静音状态下浏览。
2. **点击率为王**：视频 Pin 的点击率是推荐第一信号。
3. **保存 (Save) 是核心互动**：平台算法偏好能鼓励保存、点击和链接互动的视频。

#### 发布优化清单
- 强制叠加醒目的大号文字标题（占画面 1/4 到 1/3）。
- 选择对比最强烈的画面作为 Pin 封面。
- 标题和描述包含 2-3 个长尾搜索关键词。
- 双镜头视频的每个镜头都应有独立文字说明。

## 三、跨平台通投策略 (v2.4 更新)

### 3.1 统一素材标准
- 母版视频统一按 **9:16 (1080x1920)**、**15 秒**、**双镜头结构 (0-7s + 8-15s)** 制作。
- 音频同步指令以 TikTok 为基准添加，其他平台自动兼容。

### 3.2 平台差异化发布清单

| 发布要素 | TikTok | YouTube Shorts | Reels (IG/FB) | Pinterest |
| :--- | :--- | :--- | :--- | :--- |
| **标题风格** | 口语化 + 引导评论/收藏 | SEO 优化 + 悬念词 | 价值陈述 + 信任建立 | 关键词堆叠 + 灵感描述 |
| **标签数量** | 3-5 个 | 3-5 个长尾 | 2-3 个垂直 | 5-10 个关键词 |
| **音乐** | 必选热门音 | 可选 | 推荐版权库 | 不重要 |
| **文字叠加** | 适量 | 适量 | 推荐原生工具 | 必须加粗大字 |
| **发布频率** | 每日 1-3 条 | 每日 1-2 条 | 每周 3-5 条 | 每周 5-10 条 |
| **双镜头节奏** | 标准 0-7s + 8-15s | 标准 | IG 标准 / FB 略慢 | 每镜头加字幕 |

### 3.3 避免跨平台禁忌
- **水印问题**：任何含 TikTok 水印的视频不要直接上传至 Reels 或 Spotlight，会被严重限流。
- **音乐版权**：TikTok 热门音乐在 Reels 上可能无版权，发布前检查 Instagram 音乐库。
- **转场兼容**：快速摇摄 (Whip pan) 在所有平台均兼容，是首选转场方式。
