### 🧵 Threads（v2.14 新增）

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16) 或 1080x1080 (1:1) |
| 时长限制 | 15-60s（推荐 15-30s） |
| 文件大小 | ≤500MB |
| 帧率 | 30fps |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |

#### AIGC 合规要求
- Threads 继承 Meta 的 AI 标签政策，发布时需勾选 **"Made with AI"**。

#### 2026 算法核心信号
1. **Reps 互通加权**：Threads 视频可同步推送到 IG Reels，获得双平台流量，是 Meta 生态内零成本增量的关键渠道。
2. **文本优先分发**：与 Reels 偏重视觉不同，Threads 的帖子文案长度和互动（回复、引用、转帖）直接影响推荐权重。**纯视频无文案会被严重限流**。
3. **话题标签权重低**：Threads 不依赖标签分发，依赖内容文本相关性和用户兴趣匹配。标签仅作为内容分类辅助。
4. **实时性优先**：新闻/热点/趋势类内容获得额外曝光加权，与 Twitter/X 类似。

#### Threads 内容策略
| 策略点 | 具体做法 |
| :--- | :--- |
| **文案前置** | 视频下方必须配一段 50-200 字符的文案，引导用户互动。文案应包含：痛点共鸣 + 产品价值 + 提问引导。 |
| **热点绑定** | 发布前搜索 Threads 当日 Trending Topics，将产品与热点自然关联，提升文本相关性推荐。 |
| **回复驱动** | 结尾用开放式问题，如 "Would you try this?" 或 "What's your go-to pan?"，激发回复。高回复率帖子会被持续推荐。 |
| **多格式发布** | 同一视频同时发 Threads（9:16）+ IG Reels，文案可复用但建议微调（Threads 更偏文本，Reels 更偏视觉）。 |
| **跨平台联动** | 在 Threads 帖文中 @ 自己的 IG 账号，或在 Reels 评论区引导用户去 Threads 看完整文案，形成流量循环。 |

#### Threads 发布文案模板
```
[痛点共鸣] Tired of scrubbing pans for 20 minutes after cooking?
[产品价值] This uncoated cast iron wok slides eggs like butter and cleans in seconds.
[提问引导] Would you switch to uncoated cookware? 👇
[CTA] Link in bio if you want to check it out.
```

#### Threads 避坑清单
| ❌ 错误做法 | ✅ 正确做法 |
| :--- | :--- |
| 纯视频无文案 | 必须配 50-200 字符文案，包含提问引导 |
| 过度使用标签（#fyp #viral） | 1-2 个标签即可，或不加标签，依赖文本分发 |
| 复制粘贴 IG Reels 文案 | Threads 文案应更长、更偏“讨论”，Reels 文案偏“展示” |
| 忽略评论区回复 | 发布后 1 小时内积极回复评论，回复量影响推荐 |


### 🍋 Lemon8（v2.14 新增）

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16) 或 1080x1350 (4:5) |
| 时长限制 | 15-60s（推荐 30s） |
| 文件大小 | ≤500MB |
| 帧率 | 30fps |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |

#### AIGC 合规要求
- Lemon8 暂未强制 AI 标签，但平台强调“真实分享”调性。建议主动标注 **"AI-assisted content"** 或 **"Created with AI tools"**，以增强品牌透明度，避免用户举报。

#### 2026 算法核心信号
1. **种草意图匹配**：Lemon8 的核心是“搜索+推荐”混合。用户搜索过相关品类（如 "cast iron pan"），系统会优先推荐该类目的视频和图文。**标题关键词优化至关重要**。
2. **图文视频混合推荐**：视频与图文笔记在推荐池中同权竞争，这意味着封面图和标题对视频的点击率同样重要。一条优质视频配上一个低质封面，可能被图文笔记碾压。
3. **收藏权重极高**：类似小红书，收藏（Save）是衡量内容“有用程度”的核心信号。视频中必须明确引导用户收藏（如 "Save for your next meal prep"）。
4. **本土化内容优先**：日语、英语、泰语等内容分别有独立的推荐池。用本地语言创作是获得推荐的前提。机器翻译内容会被识别并降权。

#### Lemon8 内容策略
| 策略点 | 具体做法 |
| :--- | :--- |
| **标题关键词堆叠** | 标题必须包含品类词+核心卖点词+场景词。例如："$12 Cast Iron Pan That Changed My Cooking | No Oil, No Stick, Perfect for Small Kitchen"。 |
| **封面即转化** | 视频封面必须是一张高质量产品特写图，并叠加 1-2 个大字卖点（如 "ZERO OIL"）。Lemon8 用户先看封面和标题，再决定是否点击视频。 |
| **长尾标签策略** | 使用 5-10 个长尾标签，如 #castironpan #budgetcookware #smallkitchenhacks #nontoxicliving。Lemon8 的标签对搜索排名仍有较高影响。 |
| **评论区置顶引导** | 发布后立即置顶一条评论，直接放购买链接（如 huopan.com）或 "🔗 Link in bio"。Lemon8 对评论区外链限制较松。 |
| **图文+视频双发** | 同一产品建议同时发布：①一条视频笔记（展示过程）+ ②一条图文笔记（静态图+详细文案），双格式覆盖不同阅读偏好的用户。 |

#### Lemon8 市场差异化策略
| 市场 | 语言 | 内容偏好 | 音乐风格 | 转化路径 |
| :--- | :--- | :--- | :--- | :--- |
| 美国 | 英语 | 实用教程、性价比种草、租房党/小户型 | 轻快 pop、lofi beats、trending | bio 链接为主 |
| 日本 | 日语 | 美学细节、生活方式感、收纳整理 | 治愈系钢琴、acoustic、city pop | 楽天市場、Amazon |
| 东南亚 | 泰语/英语 | 价格敏感、实用性强、家庭场景 | 本土流行、快节奏、Luk thung | Shopee、Lazada |

#### Lemon8 避坑清单
| ❌ 错误做法 | ✅ 正确做法 |
| :--- | :--- |
| 硬广风格（直接喊价、链接） | 必须伪装成“好物分享”，用第一人称真实体验口吻 |
| 忽略封面设计 | 封面必须有产品特写+大字卖点，静态图点击率决定推荐池 |
| 只发视频不发图文 | 图文笔记同样能获得高曝光，且制作成本更低 |
| 机器翻译标题和文案 | 必须人工适配本土语言，机器翻译会被降权 |
| 标签使用过少 | 使用 5-10 个长尾标签，增加搜索曝光 |
