## 二、平台详细规格与算法深度解析

### 🎬 TikTok

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16)，最低 540x960 |
| 帧率 | 24/25/30fps (推荐 24fps) |
| 编码格式 | H.264 或 H.265 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3 Mbps (推荐 6-8 Mbps) |
| 色彩空间 | Rec. 709 |

#### AIGC 合规要求
- TikTok 要求对 AI 生成内容进行标签披露。发布时需勾选 **“AI-generated content”** 选项。
- 未标注可能导致内容被下架或限流。

#### 2026 算法核心变化
1. **小批量测试分发**：新视频发布后，首先进入约 **200-500 人的小批量测试池**（兴趣匹配受众）。测试池的反馈（完播率、收藏、分享）决定能否进入下一级流量池。**不是“粉丝优先”，而是“兴趣匹配优先”**。
2. **垂直领域强化（新增）**：2026 年算法会自动识别视频主题标签，并优先推送给对该领域有持续观看行为的用户。为确保算法精准识别，**前 5 秒内必须通过口述+字幕双重强化核心主题词**（例如清洁产品出现“cleaning hack”，收纳产品出现“organization”，美妆产品出现“nail tutorial”）。搜索权重中，语音关键词权重最高，屏幕文字次之，标题描述相对较弱。
3. **算法更新频率提升至月度**：TikTok 算法已从季度迭代升级为**月度迭代**，创作者需持续关注平台动态并及时调整策略。
4. **互动权重重塑**：收藏(Save)和分享(Share)权重首次超过点赞，是最重要的深度互动信号。
5. **TikTok 搜索成为核心信号**：短视频已成为 Z 世代首选搜索引擎，超过 50% 的年轻族群将 TikTok 视为搜索工具。标题、字幕中的关键词成为核心流量入口。
6. **完播率与重播率**：完播率门槛提升至70%，重播率超20%被视为优秀水平。

#### TikTok SEO 策略
| 优化点 | 具体做法 |
| :--- | :--- |
| **语音关键词** | 视频前 5 秒口述核心主题词（权重最高） |
| **字幕关键词** | 视频内嵌字幕包含产品名称和核心卖点词，可被搜索索引（权重次之） |
| **标题关键词** | 包含 1-2 个核心搜索关键词，放在标题前部（权重较弱） |
| **话题标签** | 3-5 个垂直标签，其中 1 个为长尾关键词标签 |
| **评论区置顶** | 置顶评论中包含核心关键词，增强搜索相关性 |

#### 发布优化清单
- 标题含 1-2 个核心关键词 + 1 个热门标签。
- **前 5 秒口述+字幕双重强化核心主题词**，帮助算法精准识别垂直领域。
- 评论区置顶引导互动，设计深度评论问题。
- 优先引导**收藏(Save)** 和**社交货币分享(Share)**。
- 背景音乐选择 Trending 榜单前 20，确保画面与节拍同步。
- **密切关注月度算法更新**，及时调整策略。
- **发布时务必勾选“AI-generated content”标签**。


### ▶️ YouTube Shorts

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16) |
| 帧率 | 24/30/60fps |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3.5 Mbps |
| 音频 | AAC 或 MP3，≥128kbps |

#### AIGC 合规要求
- YouTube 要求对经过 AI 修改或生成的内容进行披露。发布时需勾选 **“Altered content”** 选项。
- 未披露可能导致视频被移除或频道被处罚。

#### 2026 算法核心变化
1. **划走率 (Swipe Away Rate) 是关键负向指标**：算法对前 2-3 秒内被用户划走的视频极度敏感。划走率过高将严重限制推荐。**前 2 秒必须制造新奇反应**。
2. **多平台分享加权（新增）**：2026 年 YouTube Shorts 算法新增“多平台分享”信号。将 Shorts 链接分享至 X/Twitter、Facebook 等社群平台，可获得额外推荐加权。建议在发布后主动引导观众跨平台分享。
3. **搜索成为关键流量来源**：2026年1月，YouTube 新增 Shorts 专属搜索过滤器。
4. **重播率是推荐核心**：完成率达到70%以上的 Shorts 会获得激进推广。
5. **长视频联动**：Shorts 可关联长视频，相互反哺。

#### 发布优化清单
- **前 2 秒必须抓人**，避免任何缓慢空镜，降低划走风险。
- 标题必须做 SEO：核心关键词前置，约40字符内。
- 描述区前 100 字符包含核心关键词，并添加 3-5 个相关标签。
- 首镜头应包含可被搜索索引的关键词视觉元素。
- **引导观众将视频分享至 X/Twitter 或 Facebook**（可在描述区或评论区添加引导语），获取多平台分享加权。
- 视频末尾 2 秒添加“订阅”动画按钮。
- **发布时务必勾选“Altered content”披露选项**。


### 📷 Instagram Reels

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16) |
| 帧率 | 30fps 最低 |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3.5 Mbps |
| 音频 | 推荐使用 Instagram 音乐库内版权音乐 |

#### AIGC 合规要求
- Meta 要求对 AI 生成或修改的内容进行标签披露。发布时需勾选 **“Made with AI”** 标签。
- 未标注可能导致内容被降权或移除。

#### 2026 算法核心变化
1. **“真实兴趣” (UTIS) 模型**：Meta 引入用户真实兴趣调查，不再只依赖点赞、观看时长等行为信号。
2. **当日发布优先（深度策略）**：Meta 算法对当日发布的新 Reels 提供**高达 50% 的额外分发权重**。**建议将多个视频分散到不同日期发布（每日 1-3 条），而非集中在一天发布**，以最大化获取“当日发布优先”的流量加权。频繁发布（如一天发5-10条）可能导致账号被判定为低质。
3. **搜索优先 (Instagram SEO)**：搜索排名受账号名称、Bio、Caption、Reels 字幕影响，Hashtag 影响力下降。

#### 发布优化清单
- **发布频率**：每日 1-3 条，**分散到不同日期发布**，而非集中在一天。保持内容新鲜度以获取当日发布优先推荐。
- 前 3 秒亮明品牌/产品名，建立信任预期。
- 使用 Instagram 内置文字工具添加字幕，提升搜索可见性。
- 避免过度夸张的病毒式文案，强调真实价值。
- **发布时务必勾选“Made with AI”标签**。

3. **无公开评论区**：互动信号主要依赖重播、点赞和分享。

#### 发布优化清单

- 时长控制在 15-25 秒。
- 去除所有水印后再上传。


### 📌 Pinterest

#### 技术规格
| 参数 | 要求 |
| :--- | :--- |
| 分辨率 | 1080x1920 (9:16)，也支持 1080x1080 (1:1) |
| 帧率 | 25/30fps |
| 编码格式 | H.264 |
| 容器格式 | MP4 / MOV |
| 比特率 | ≥3 Mbps |
| 音频 | **不重要**，默认静音播放 |

#### 2026 算法核心
1. **视觉自解释力**：90% 用户在静音状态下浏览。
2. **点击率为王**：视频 Pin 的点击率是推荐第一信号。
3. **保存 (Save) 是核心互动**：平台算法偏好能鼓励保存、点击和链接互动的视频。
4. **多格式创作者加权（2026年3月新增）**：同时发布 Idea Pins、标准 Pins 和视频 Pins 的创作者可获得额外曝光加权。

#### 发布优化清单
- 强制叠加醒目的大号文字标题（占画面 1/4 到 1/3）。
- 选择对比最强烈的画面作为 Pin 封面。
- 标题和描述包含 2-3 个长尾搜索关键词。
- **多格式发布**：建议将视频同时以 Idea Pin 和视频 Pin 形式发布，获取额外加权。


## 三、跨平台通投策略 (v2.7 更新)

### 3.1 统一素材标准
- 母版视频统一按 **9:16 (1080x1920)**、**15 秒**、**品类匹配多镜头模板（3-4 镜头）** 制作。
- 音频同步指令以 TikTok 为基准添加，其他平台自动兼容。
- **前 5 秒口述+字幕双重强化核心主题词**，确保 TikTok 垂直领域信号和 SEO 优化同步完成。

### 3.2 平台差异化发布清单

| 发布要素 | TikTok | YouTube Shorts | Reels (IG/FB) | Pinterest |
| :--- | :--- | :--- | :--- | :--- |
| **标题风格** | 口语化 + 引导评论/收藏 | SEO 优化 + 悬念词 | 价值陈述 + 信任建立 | 关键词堆叠 + 灵感描述 |
| **SEO 重点** | 语音关键词(前5秒) > 字幕 > 标题 | 标题+描述关键词 | 字幕关键词 | 标题+描述关键词 |
| **标签数量** | 3-5 个 | 3-5 个长尾 | 2-3 个垂直 | 5-10 个关键词 |
| **音乐** | 必选热门音 | 可选 | 推荐版权库 | 不重要 |
| **文字叠加** | 适量 | 适量 | 推荐原生工具 | 必须加粗大字 |
| **发布频率** | 每日 1-3 条 | 每日 1-2 条 | **每日 1-3 条（分散发布）** | 每周 5-10 条 |
| **特别关注** | **垂直领域信号+突破200-500人测试池** | **降低前2秒划走率+多平台分享** | **当日发布优先** | **多格式发布加权** |
| **AIGC标签** | AI-generated content | Altered content | Made with AI | 建议主动标注 |

### 3.3 避免跨平台禁忌
- **水印问题**：任何含 TikTok 水印的视频不要直接上传至 Reels 或 Spotlight，会被严重限流。
- **音乐版权**：TikTok 热门音乐在 Reels 上可能无版权，发布前检查 Instagram 音乐库。
- **转场兼容**：快速摇摄 (Whip pan) 在所有平台均兼容，是首选转场方式。
