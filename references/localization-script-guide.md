# 多语言本土化脚本生成指南 (Localization Script Guide) v1.0

> **用途**：将单条英语脚本转化为多语言版本，适配不同市场的语言习惯、文化偏好和电商生态，实现“一条素材，全球复用”。
> **触发时机**：在 `SKILL.md` 阶段 2 脚本结构化完成后，用户可选择执行多语言生成（阶段 2.5）。
> **创建版本**：v1.0 (2026.05) —— 初始版本，覆盖 7 种主要语言市场。


## 一、本土化核心原则

| 原则 | 说明 | 示例 |
| :--- | :--- | :--- |
| **非直译，重情绪** | 根据目标语言习惯重写，保留情绪冲击力而非字面含义 | "This feels illegal to know" → 日语「これ知ったら戻れない」（知道了就回不去了） |
| **文化适配** | 话术风格必须符合目标市场审美 | 美国直接夸张，日本含蓄细腻，韩国潮流犀利 |
| **节奏匹配** | 目标语言的音节数需与原版时长匹配，确保口播与画面同步 | 英语 7 音节 → 日语需控制在 10-12 音节 |
| **保留钩子结构** | 钩子类型在各语言市场均有效，只需替换话术，不改变剧本结构 | 认知失调型、视觉奇观型等 7 类钩子通用 |
| **转化路径本土化** | 不同市场的主流电商平台不同，CTA 需适配 | 美国 TikTok Shop，日本楽天，东南亚 Shopee |


## 二、各市场本土化策略速查

### 🇺🇸 美国市场（英语）

| :--- | :--- |
| **话术风格** | 直接、夸张、第一人称、制造 FOMO |
| **常用情绪词** | insane / crazy / wild / illegal to know / game changer |
| **音乐偏好** | Trending audio、快节奏 pop、Hip-hop beats |
| **转化路径** | TikTok Shop 直链（优先）、评论区 bio 链接 |
| **AIGC 标签** | "AI-generated content" |

## 三、多语言脚本生成工作流

```
阶段 2 完成英语脚本
        ↓
用户选择目标市场（可多选，如：日本 + 西班牙语 + 泰语）
        ↓
AI 执行本土化翻译：
    - 保留钩子结构
    - 调整话术风格（对照 §二）
    - 适配音乐偏好
    - 调整转化路径
        ↓
输出「多语言脚本对照表」（模板见 §四）
        ↓
用户选择语言版本 → 进入阶段 3 提示词工程
```

> **成本说明**：每增加一门语言，增加一次 Prompt 工程时间（约 2-3 分钟），视频生成成本不变（可复用同一提示词的视觉部分）。

## 五、本土化自检清单

翻译完成后，逐项检查：

- [ ] **钩子类型是否保留**？（认知失调/视觉奇观/故事型等不应改变）
- [ ] **话术风格是否符合目标市场审美**？（对照 §二 各市场风格）
- [ ] **音节数是否与原版匹配**？（±2 音节，确保口播与画面同步）
- [ ] **音乐建议是否符合目标市场偏好**？（对照 §二 音乐偏好）
- [ ] **转化路径是否符合目标市场电商生态**？（TikTok Shop / 亚马逊 / 楽天 / Shopee）
- [ ] **是否有文化禁忌词汇**？（如日本避免「死ぬ」「切腹」；中东避免宗教敏感词）
- [ ] **AIGC 标签是否已提醒**？（各市场标签名称不同，对照 §二）


## 六、常见问题

**Q1：如果目标市场语言我不会，怎么验证翻译质量？**
A：建议使用 DeepL + ChatGPT 双重校验，或雇佣 Fiverr 上的母语者进行校对（成本约 $5-10/版）。

**Q2：多语言版本的视频画面需要重新生成吗？**
A：不需要。纯产品展示类视频（如锅具、清洁）画面通用，只需替换口播配音和字幕。若有人物出镜，建议使用 HeyGen 等工具进行口型同步。

**Q3：哪些市场建议优先覆盖？**
A：优先级排序：① 美国（最大 TikTok 电商市场）→ ② 日本/韩国（高客单价）→ ③ 西班牙语（覆盖多国）→ ④ 东南亚（增速快）。


## 七、与 Skill 工作流的集成
