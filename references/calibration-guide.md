# 评分校准指南 (Calibration Guide) v1.0

> **用途**：将 Prompt 预评估的分数与实际视频表现进行对照，通过轻量级数据分析优化评分表权重，实现“数据驱动校准”。
> **适用对象**：已使用 Skill 生成至少 20-30 条视频，希望系统性优化评分标准的用户。
> **创建版本**：v1.0 (2026.05) —— 初始版本，提供手动校准方法及自动化脚本。

---

## 一、为什么需要校准？

当前 Prompt 预评估的权重（H:30, A:25, V:20, U:15, D:10）是基于经验和数学推理设定的。但不同品类、不同市场、不同时期，各维度的实际重要性可能不同。

例如：
- 美国市场声音钩子（A）的权重可能远高于垂直领域信号（V）
- 清洁品类原生感（U）的影响可能低于锅具品类

通过收集实际数据并分析各维度得分与实际表现的相关性，可以发现**你的品类和市场中真正重要的维度**，并调整权重以更精准地预测爆款概率。


## 二、数据收集：结构化追踪模板

### 2.1 单条视频数据记录格式

## 三、手动校准方法（Excel / Google Sheets）

### 步骤 1：计算各维度与目标指标的相关系数

1. 在 Google Sheets 中，使用 `=CORREL()` 函数计算每个维度得分与实际表现的相关系数。
2. **推荐目标指标**：`completion_rate`（完播率）是最直接的视频质量信号，优先使用。也可用 `save_rate + share_rate` 的组合指标。

**示例公式**（假设 H 分数在 D 列，完播率在 J 列，数据从第 2 行到第 51 行）：
```
=CORREL(D2:D51, J2:J51)
```

对 H/A/V/U/D 五个维度分别计算相关系数。

### 步骤 2：将相关系数转换为新权重

1. 相关系数范围为 [-1, 1]。负值或接近 0 的维度说明与表现无关，权重应降低或设为 0。
2. 将所有**正相关系数**求和，每个维度的新权重 = 该维度相关系数 / 正相关系数之和。

**示例**：

| 维度 | 相关系数 r | r（仅取正值） | 新权重（归一化） |
| :--- | :--- | :--- | :--- |
| H | 0.45 | 0.45 | 0.45 / 1.00 = **45%** |
| A | 0.30 | 0.30 | 0.30 / 1.00 = **30%** |
| V | 0.05 | 0.05 | 0.05 / 1.00 = **5%** |
| U | 0.20 | 0.20 | 0.20 / 1.00 = **20%** |
| D | -0.05 | 0（丢弃） | **0%** |
| **合计** | — | **1.00** | **100%** |

> **解读**：在这个示例中，钩子强度(H)和声音策略(A)是最重要的驱动因素，去AI味指令(D)与完播率几乎无关，可以移除或降低权重。

### 步骤 3：应用新权重

1. 将新权重更新到 `evaluation-rubric.md` 的评分表中（建议保留一份原版备份）。
2. 继续收集数据，每新增 20 条视频重复一次校准。


## 四、自动化校准脚本（Python，可选）

如果熟悉 Python，可使用以下脚本自动计算相关系数和新权重。

```python
import pandas as pd

# 1. 读取数据（假设为 CSV 格式）
df = pd.read_csv('calibration_data.csv')

# 2. 定义维度列和目标列
dimensions = ['H_score', 'A_score', 'V_score', 'U_score', 'D_score']
target = 'completion_rate'  # 可替换为 save_rate 等

# 3. 计算相关系数
correlations = {}
for dim in dimensions:
    r = df[dim].corr(df[target])
    correlations[dim] = r if r > 0 else 0  # 负相关或零相关置零

# 4. 归一化为权重
total = sum(correlations.values())
new_weights = {dim: round(corr / total, 3) for dim, corr in correlations.items()} if total > 0 else {}

print("相关系数:", correlations)
print("新权重:", new_weights)
```

**使用步骤**：
1. 将 Google Sheets 导出为 CSV 文件，命名为 `calibration_data.csv`。
2. 安装 Python 及 pandas 库：`pip install pandas`。
3. 将脚本保存为 `calibrate.py`，与 CSV 文件同目录运行。
4. 根据输出的新权重更新评分表。


## 五、校准后的持续迭代

1. **定期校准**：建议每积累 30-50 条新数据后重新运行一次校准。
2. **分品类/分市场校准**：如果数据量足够（每个子集 ≥30 条），可按品类或市场分别校准，发现更精细的权重差异。
3. **权重平滑**：如果新旧权重差异过大，可采用渐进式调整（如新权重 = 0.7×旧权重 + 0.3×回归权重），避免剧烈波动。


## 六、常见问题

**Q1：相关系数都很低（<0.2）怎么办？**
A：可能原因：1) 样本量不足，继续收集数据；2) 实际视频表现受外部因素影响大（如发布时间、竞品环境），校准仍有参考价值但需谨慎；3) 评分维度可能遗漏了重要变量（如趋势热度），考虑增加新维度。

**Q2：某个维度相关系数为负，是否应该删除？**
A：理论上负相关意味着该维度高分反而降低表现。如果样本量足够且持续为负，可以将其权重降为 0 或从评分表中移除。但建议先检查是否存在数据录入错误或异常值。

**Q3：校准后权重变化很大，是否立即应用？**
A：建议先在后续 5-10 条视频中测试新权重的预测效果，对比新旧权重的预评估分与实际表现的相关系数，确认优化后再正式替换。
