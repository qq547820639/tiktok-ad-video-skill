# 带货视频自查清单 (Self-Check Checklist) v2.10

> **用途**：视频发布前逐项核对，确保最大化爆款概率。
> **使用方式**：每生成一条视频，对照本清单检查，全部通过后再发布。
> **更新版本**：v2.10 (2026.05) —— 新增叙事型视频自查清单，区分直给型与叙事型检查项。


## 一、通用检查项（所有视频类型适用）

### 1.1 发布前最后确认

| 检查项 | ✅ | ❌ | 备注 |
| :--- | :--- | :--- | :--- |
| 1. 是否勾选了平台的 **AIGC 标签**？ | | | TikTok: AI-generated；Meta: Made with AI；YouTube: Altered content |
| 2. 是否设置了 **7 天后回看数据** 的日历提醒？ | | | 数据归因，驱动迭代 |
| 3. 是否排查了 **2026 Q2 违禁词**？ | | | 医疗健康类、绝对化极限词、虚假承诺类 |


## 二、直给型广告自查清单（15 秒）

### 2.1 前 3 秒钩子检查（权重最高）

| 检查项 | ✅ | ❌ | 备注 |
| :--- | :--- | :--- | :--- |
| 1. 首帧是否包含“认知失调”或“视觉奇观”？ | | | 让人本能停止划动 |
| 2. 前 3 秒是否有 **声音钩子**（ASMR/音效/环境音）？ | | | 声音优先级 > 画面 |
| 3. 口播是否在 **第 3 秒之后** 才进入？ | | | 画面先讲故事，口播后解释 |
| 4. 用户看完 3 秒会不会想问一个问题？（如“没放油？”“她真回头了？”） | | | 制造好奇 |

### 2.2 钩子类型选型检查（按品类对照）

| 品类 | 推荐钩子 | 避免的钩子 | 当前视频是否匹配？ |
| :--- | :--- | :--- | :--- |
| **香水/美妆/服饰** | 场景共鸣 POV / 社交反馈 | 价格对比、成分科普 | [✅/❌] |
| **锅具/厨房用品** | 听觉解压 ASMR / 奇观展示 / 故事型 | 健康恐吓、温和展示 | [✅/❌] |
| **3C/家电（高客单价）** | 痛点对比 / 技术下放 | 纯参数罗列 | [✅/❌] |
| **鞋服/日用百货** | 百搭展示 / 极限性价比反差 | 功能平淡叙述 | [✅/❌] |
| **清洁用品** | 认知失调型（污渍瞬间消失） | 温和展示 | [✅/❌] |
| **食品饮料** | 视觉奇观型 / 故事型（Adulting Win） | 成分罗列、健康宣称 | [✅/❌] |
| **收纳用品** | 极简结果型 | 静态展示 | [✅/❌] |

### 2.3 脚本结构检查

| 检查项 | ✅ | ❌ | 备注 |
| :--- | :--- | :--- | :--- |
| 1. 镜头数量是否符合品类模板（3-4 镜头）？ | | | 功能效果型 4 镜头，极简结果型 3 镜头 |
| 2. 是否植入了 **复播彩蛋**（信息差/节奏中断/双重结局）？ | | | 提升复播率 |
| 3. 是否包含了 **收藏引导** 话术？ | | | 如“Save for your next ___” |
| 4. 是否包含了 **社交货币分享** 话术？ | | | 展现品味/展示专业/圈层归属 |
| 5. 前 5 秒是否 **口述+字幕双重强化** 核心主题词？ | | | 帮助算法精准识别垂直领域 |

### 2.4 字幕与大字报检查

| 检查项 | ✅ | ❌ | 备注 |
| :--- | :--- | :--- | :--- |
| 1. 每个核心卖点是否配一行大字报？ | | | 不要堆砌，每屏只一个核心信息 |
| 2. 每行大字报停留时间是否 ≥ 1.5 秒？ | | | 确保观众可读完 |
| 3. 大字报是否放在画面安全区内？ | | | 避免被 TikTok UI 遮挡 |
| 4. 静音状态下是否完全可懂？ | | | Pinterest/Reels 大量静音浏览场景 |

### 2.5 数据反馈迭代动作（直给型）

| 数据表现 | 迭代动作 | 执行状态 |
| :--- | :--- | :--- |
| 播放量 < 200，前 3 秒留存 < 40% | **换钩子类型**（对照品类选型表） | [ ] |
| 播放量 500-2000，完播率 30-50% | **微调卖点顺序** + **强化评论区互动** | [ ] |
| 播放量 > 2000，完播率 > 50% | **复制脚本结构**，换品再做一条 | [ ] |


## 三、叙事型软广自查清单（45-60 秒）—— v2.10 新增

> **适用场景**：美国/欧洲市场，连续 3 条直给型广告播放量 < 500 时切换至叙事型。

### 3.1 钩子与悬念检查

| 检查项 | ✅ | ❌ | 备注 |
| :--- | :--- | :--- | :--- |
| 1. 前 3 秒是否建立了 relatable 的困境/好奇心？ | | | 如“worked 10 hours and this is all that's left in my fridge” |
| 2. 前 3 秒是否有声音钩子（环境音/ASMR，不要人声开场）？ | | | 叙事型更依赖环境音营造氛围 |
| 3. 观众是否有理由看到第 30 秒？（悬念维持检查） | | | 12-22 秒是悬念增强的关键段落 |
| 4. 观众是否有理由看到最后？（情绪释放检查） | | | 结尾必须有明确的情绪收束或彩蛋 |
| 5. 叙事模板是否与产品匹配？ | | | Mystery Object（首选）/ Adulting Win / Before You Judge |

### 3.2 产品植入检查（零提及纪律）

| 检查项 | ✅ | ❌ | 备注 |
| :--- | :--- | :--- | :--- |
| 1. 视频内是否 **零提及** 品牌名？ | | | 包括口播、字幕、画面文字 |
| 2. 视频内是否 **零提及** 价格？ | | | 价格对比、价格揭示全部后置 |
| 3. 视频内是否 **零提及** 购买引导（link/buy/shop/order）？ | | | 所有 CTA 后置到评论区 |
| 4. 视频内是否 **零展示** 购买页面截图？ | | | 包括手机屏幕上的商品页 |
| 5. 产品是否作为“故事道具”自然出现？ | | | 不刻意特写品牌，但可特写质感 |
| 6. 产品效果展示是否足够有说服力（不需要语言解释）？ | | | 画面自己会说话 |

### 3.3 评论区准备（叙事型核心）

| 检查项 | ✅ | ❌ | 备注 |
| :--- | :--- | :--- | :--- |
| 1. 是否准备了 2-3 个小号评论？ | | | 发布后 3 分钟内发布 |
| 2. 小号评论是否覆盖“好奇心 + 效果惊叹 + 社交分享”三种类型？ | | | 模板见 `platform-specs.md` §4.3 |
| 3. 主号回复话术是否准备好了？ | | | 自然植入产品信息，不用客服语气 |
| 4. bio 链接是否已经设置好？ | | | 确保用户点击后能直达商品页 |

### 3.4 分段生成检查（Seedance 2.0 专用）

| 检查项 | ✅ | ❌ | 备注 |
| :--- | :--- | :--- | :--- |
| 1. 是否按 4 段分段生成？（50 秒 = 4 段 × 15 秒） | | | 第 1、3 段 Standard，第 2、4 段 Fast |
| 2. 每段提示词是否锁定了 **一致性要素**？ | | | 主角外貌、场景、色调、光影必须一致 |
| 3. 各段之间的转场是否自然？ | | | 拼接处无明显跳帧或色调突变 |
| 4. 总时长是否控制在 45-60 秒？ | | | 过短悬念不足，过长完播率下降 |

### 3.5 数据反馈迭代动作（叙事型专用）

| 数据表现 | 迭代动作 | 执行状态 |
| :--- | :--- | :--- |
| 播放量 < 200，前 3 秒留存 < 30% | **换钩子类型**（对照叙事模板 A-E） | [ ] |
| 播放量 < 200，前 3 秒留存 > 30% 但完播率 < 20% | **中间段落拖沓**，剪掉 5-10 秒 | [ ] |
| 播放量 200-500，评论少但分享多 | 内容好但互动引导不足，**强化评论区运营** | [ ] |
| 播放量 200-500，评论多但分享少 | 内容有趣但不够 shareable，**增加社交货币元素** | [ ] |
| 播放量 500-2000，完播率 > 40% | **内容验证成功**，用同模板做第二条 | [ ] |
| 播放量 500-2000，完播率 < 40% | 前段钩子有效但后段塌方，**优化后半段节奏** | [ ] |
| 播放量 2000-10000 | **复制脚本结构**，换品类再做 / 同品类换场景再做 | [ ] |
| 播放量 > 10000 | **立刻做第二条**（评论区会告诉你第二条拍什么） | [ ] |


## 四、长期内容矩阵建议

不要指望一条视频爆单。每个产品至少准备 **3 个不同钩子方向的视频** 同时跑：

**直给型矩阵**：
- **视频 A**：情绪/场景钩子（负责拉播放量）
- **视频 B**：功能/暴力测试钩子（负责建立信任）
- **视频 C**：性价比/价格对比钩子（负责转化）

**叙事型矩阵**（美国市场）：
- **视频 A**：Mystery Object（悬念驱动，拉评论）
- **视频 B**：Adulting Win（情绪共鸣，拉完播和分享）
- **视频 C**：Plot Twist（反转型，拉转化）

三条跑完，看数据，留下表现最好的方向，再复制 5-10 条变体。

---

**记住**：前 3 秒声音钩子比画面更重要；钩子选错满盘皆输；美国市场播放量卡 300 就切叙事型；数据是最好的导演。
