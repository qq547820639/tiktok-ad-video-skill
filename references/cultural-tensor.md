# 跨文化传播张量模块 (v4.0-Adaptive · New)

> **模块编号**: CT-001  
> **加载条件**: 目标市场为非美国英语区，或用户明确要求跨文化适配时加载  
> **理论框架**: Hofstede 文化维度理论 + Schwartz 基本价值观  
> **v4.0 角色**: 将文化维度编码为可计算向量，作为 Prompt 生成器的协变量输入，解决跨市场分发的方差风险

---

## 一、Hofstede 文化维度速查

| 维度 | 缩写 | 高值含义 | 低值含义 |
|:---|:---|:---|:---|
| 权力距离 | PDI | 接受等级差异，尊重权威 | 追求平等，挑战权威 |
| 个人主义 | IDV | 个人目标优先，独立表达 | 集体目标优先，群体和谐 |
| 男性气质 | MAS | 竞争、成就、物质成功 | 合作、生活品质、关怀 |
| 不确定性规避 | UAI | 偏好规则、确定性、焦虑未知 | 接受模糊、灵活应变 |
| 长期导向 | LTO | 节俭、坚持、为未来投资 | 尊重传统、履行社会义务 |
| 放纵 | IVR | 允许满足欲望，追求快乐 | 克制欲望，严格社会规范 |

## 五、视觉/听觉文化禁忌清单

| 元素 | 禁忌市场 | 替代方案 |
|:---|:---|:---|
| 夸张的冲突、翻白眼等微表情 | 日本、韩国、中东 | 改为产品物理演示，不涉及人际冲突 |
| "挑战传统/打破规矩"话术 | 日本、韩国、中东、中国 | 改为"升级版/新一代/传承中的创新" |
| 单手递物 | 韩国、日本 | 双手递物或画面焦点在物品本身 |
| 猪/酒/裸露元素 | 中东 | 完全移除，替换为植物/茶/保守着装 |
| 尖锐的声音效果（警报声、碎裂声）| 日本 | 改为柔和 POP 音或自然环境音 |
| 高饱和荧光色 | 日本、德国 | 降饱和、自然调色盘 |
| "限时抢购"紧迫感过强 | 德国、日本 | 改为"限量供应，售完即止"的温和表达 |
| 明星/网红的口语化夸张推荐 | 德国 | 改为真实用户的数据化评价 |
| 个人成就的过度炫耀 | 日本、韩国 | 改为"大家的支持让我们做到"的集体归因 |
| 直接比较竞争对手 | 德国（法律风险）| 改为"传统方式 vs 本产品"的无品牌对比 |

## 六、与动态权重引擎的对接（🆕 v4.0）

### 6.1 文化向量作为协变量

在 Prompt 生成阶段，文化向量 C 作为输入：

```
Prompt = f(Product, Hook_Type, Culture_Vector)
```

其中 `Culture_Vector` = [PDI, IDV, MAS, UAI, LTO, IVR]，标准化至 [0,1] 区间。

### 6.2 跨市场权重共享规则

- **独立权重**：不同市场的 Hook 类型权重独立维护（美国市场的 `w_hook_conflict` 与日本市场的 `w_hook_conflict` 不做跨市场平均）
- **先验借用**：当某市场样本 n < 5 时，借用文化向量相似的市场的先验参数作为冷启动 Prior
  - 相似度定义：Euclidean Distance( Culture_Vector_i, Culture_Vector_j ) < 0.3 时允许借用
- **借用衰减**：借用先验的权重随本市场样本增加而衰减，n ≥ 10 时完全停止借用

### 6.3 文化层失败的回传

当视频因文化层原因失败（如 FC-009），更新机制：
- 该市场 + 该 Hook 类型的 β_culture += 1
- 若同一市场同一文化维度连续触发 3 次失败 → 该 Hook 类型在该市场自动降级为 ❌（禁选）
- 恢复条件：本市场连续 10 条视频无文化层失败，重置为 ⚠️（可测）。
