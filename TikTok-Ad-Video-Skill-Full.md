## 第一部分：核心工作流

### 1. 核心铁律（10 条）

1. **前 3 秒定生死**：首帧必须有视觉冲突/悬念，优先 ASMR/音效（口播第 3 秒后进入）。
2. **原生感 (UGC) 优先**：看起来像真实用户分享，非精修广告。
3. **三维匹配**：钩子选择必须交叉验证品类×钩子×趋势热度（上升期🚀优先，衰减期📉警告）。
4. **原子化分段指令**：直给型提示词每段 30-60 词，单镜头单一运镜，禁止否定式负向词。
5. **先验证、后投入**：三层快速诊断未通过绝不消耗积分。
6. **收藏+社交货币分享优先**：2026 最高权重信号。
7. **垂直领域信号**：前 5 秒口述+字幕双重强化核心主题词。
8. **复播钩子必须植入**：信息差/节奏中断/双重结局。
9. **一稿通投，平台微调**：8 平台差异化发布。
10. **数据闭环**：发布后反馈数据 → 自动迭代 → 样本≥20 条触发校准。

### 7. 提示词输出规范（核心）

**原则**：直给型视频必须输出为「原子化分段指令」，每段 **30-60** 英文词（或中英文混写），**绝对不超过 100 词**。

#### 7.1 官方 6 步公式

所有提示词必须遵循：**主体/角色 → 动作/剧情 → 环境/场景 → 镜头/运镜 → 风格/氛围 → 约束/避坑**。

#### 7.2 直给型输出格式

```
[镜头1: 0-3s | 运镜+动作]
4K UHD, 48fps, [焦段]。 [运镜前缀] + [主体动作]。 [光影风格]。 [正向约束锚点词]。

[镜头2: 3-4s | 转场+微距]
[转场方式]。 4K UHD, 120fps, 100mm macro lens。 [微距动作]。 Visual sync with [音效]。 [正向约束]。
```

#### 7.3 正/负向词对照表

| ✅ 使用（正向锚点词） | ❌ 禁止（否定式负向词） |
| :--- | :--- |
| `Well-formed hands` | `No extra fingers` |
| `Stable object geometry` | `No warping` |
| `Natural facial features` | `No face distortion` |
| `Steady motion` | `No jitter` |
| `Crystal-clear` | `No blur` |
| `Sharp focus on product` | `Don't make it blurry` |

### 8. 三层快速诊断（阶段 3.5 · 量化版）

| 层级 | 检查项 | 量化标准 | 不通过动作 |
| :--- | :--- | :--- | :--- |
| 🔴 **致命层** | 钩子是否有视觉冲突？是否有声音钩子？ | 二选一：任一不满足即不通过 | **禁止提交**，退回阶段 2 |
| 🟡 **核心层** | 垂直信号是否前 5 秒口述+字幕双出现？原生感是否启用？ | 垂直信号缺失扣 5 分；原生感未启用扣 3 分 | **警告 + 微调**，用户确认后提交 |
| 🟢 **卫生层** | 是否含否定式负向词？是否超过 100 词/段？ | 含否定词自动替换；超长自动拆分 | **自动修正**，直接提交 |

| :--- | :--- |

### 17. 叙事型软广剧本指南

#### 17.1 五种叙事模板

| 模板 | 核心叙事逻辑 | 适用场景 | 视频时长 |
| :--- | :--- | :--- | :--- |
| **Mystery Object** | 困境→发现→验证→收尾 | 产品外观独特或效果反常识 | 50-55s |
| **Before You Judge** | 偏见→尝试→打脸→认输 | 产品普通但效果惊人 | 45-50s |
| **Adulting Win** | 困境→行动→成果→升华 | 22-35 岁独立生活人群 | 50-55s |
| **Secret Ingredient** | 展示结果→追问→揭秘→彩蛋 | 产品是惊艳结果的隐藏原因 | 40-50s |
| **Routine Reveal** | 开始→流程展示→细节特写→收尾 | 产品融入日常流程 | 45-55s |

#### 17.2 美国市场"零提及"植入原则

**视频内绝对不做**：展示品牌 Logo/包装、说产品名称/价格、说"link""shop""buy""order"、展示购买页面截图。

**视频内可以做**：产品自然出现在使用场景中、展示产品效果、产品外观自然入镜（不刻意特写品牌）、主角对产品效果的真实反应。

**转化全部后置到**：评论区（小号提问→主号回复）、个人主页 bio 链接、个人主页置顶视频。

#### 17.3 评论区运营 Playbook

**发布后 0-5 分钟**：用 2-3 个小号立刻发布评论，制造"自然追问"氛围：
```
小号A（好奇心型）："wait WHAT is that?? I need answers 😭"
小号B（效果惊叹型）："the way the egg just slid off... I'm watching this on repeat"
小号C（社交型）："sending this to everyone I know"
```

**主号回复话术库**：
- 用户问"what is this?" → "it's a cast iron pan! link in my bio"
- 用户问"where did you get it?" → "I put the link in my bio!"
- 用户问"does it really not stick?" → "only if you season it right! I can make a video on that if you want 👀"

**铁律**：不要复制粘贴相同回复；每条回复像朋友聊天；用回复引导更多互动；不要在视频内放任何购买引导；小号之间不要互相回复。

#### 17.4 分段生成策略

叙事型视频需要分段生成后在剪映中拼接。

| 段落 | 时长 | Seedance 参数 | 提示词重点 |
| :--- | :--- | :--- | :--- |
| 第 1 段 | 15s | Standard 模式 | 环境建立 + 钩子情绪 |
| 第 2 段 | 15s | Fast 模式 | 产品登场 + 好奇心 |
| 第 3 段 | 15s | Standard 模式 | 产品使用过程（核心画面） |
| 第 4 段 | 10-15s | Fast 模式 | 情绪收尾 |

**总成本**：约 360-408 积分（vs 全部 Standard 的 480 积分，节省 20%+）

**段落间一致性控制**：每段提示词必须锁定主角外貌、场景、光影、色调。

### 19. 实战案例库

#### 19.1 锅具/厨房用品

**✅ 成功案例 1：铸铁锅 · 视觉奇观型**
- 钩子："No oil. No butter. Just slide."
- 声音策略：前 3 秒纯 ASMR 煎蛋滋滋声，无口播

- **可复用公式**：`[无油声明] + [ASMR 音效前3秒] + [滑动物体特写] + [定格+收藏引导]`

**✅ 成功案例 2：铸铁锅 · 叙事型**

**❌ 失败案例：不粘锅 · 功能罗列型**
