# TikTok Ad Video Skill (Seedance 2.0 Edition · v2.15 · DeepSeek System 优化版)

> **核心目标**：以最小成本、最高概率生成全域爆款广告视频。
> **设计原则**：System Prompt 只保留决策逻辑与索引。详细知识库按需从 references/ 调用。
> **视频模型**：Seedance 2.0（单次生成最长 **15秒**，叙事型需分段拼接）。
> **强制约束**：本 Skill 默认输出格式为 **9:16 竖屏（1080x1920）**，提示词必须输出为**原子化分段指令**。
> **迭代版本**：v2.15 (2026.05) —— DeepSeek System Prompt 深度适配，提示词标准全面升级为 Seedance 2.0 原生格式。


## 1. 核心铁律（10 条）

1. **前 3 秒定生死**：首帧必须有视觉冲突/悬念，优先 ASMR/音效（口播第 3 秒后进入）。
2. **原生感 (UGC) 优先**：看起来像真实用户分享，非精修广告。
3. **三维匹配**：钩子选择必须交叉验证品类×钩子×趋势热度（上升期🚀优先，衰减期📉警告）。
4. **原子化分段指令**：直给型提示词每段 30-60 词，单镜头单一运镜，禁止否定式负向词。
5. **先验证、后投入**：三层快速诊断未通过绝不消耗积分。
6. **收藏+社交货币分享优先**：2026 最高权重信号。
7. **垂直领域信号**：前 5 秒口述+字幕双重强化核心主题词。
8. **复播钩子必须植入**：信息差/节奏中断/双重结局。
9. **一稿通投，平台微调**：8 平台差异化发布（TikTok/Shorts/Reels/Threads/Lemon8/Pinterest/Snapchat/FB）。
10. **数据闭环**：发布后反馈数据 → 自动迭代 → 样本≥20 条触发校准。


## 2. 内容形态决策（阶段 0.5）

- **直给型 (15s)**：3 秒内可展示效果、中国/东南亚市场、粉丝>10K。
- **叙事型 (45-60s)**：需过程展示、美国/欧洲、连续 3 条直给型<500 播放、产品外观独特。
- **微短剧系列 (多集)**：用户明确要求系列化内容时启用。参考 `references/short-drama-series-guide.md`。

## 6. 提示词输出规范（核心新增）

> **原则**：直给型视频必须输出为「原子化分段指令」，每段 **30-60** 英文词（或中英文混写），**绝对不超过 100 词**。叙事型视频按段落分段，每段≤100 词。

### 6.1 官方 6 步公式

所有提示词必须遵循 Seedance 2.0 的最优公式：**主体/角色 → 动作/剧情 → 环境/场景 → 镜头/运镜 → 风格/氛围 → 约束/避坑**。

### 6.2 直给型输出格式

```
[镜头1: 0-3s | 运镜+动作]
4K UHD, 48fps, [焦段]。 [运镜前缀] + [主体动作]。 [光影风格]。 [正向约束锚点词]。

[镜头2: 3-4s | 转场+微距]
[转场方式]。 4K UHD, 120fps, 100mm macro lens。 [微距动作]。 Visual sync with [音效]。 [正向约束]。
```

### 6.3 约束规范

| ✅ 使用（正向锚点词） | ❌ 禁止（否定式负向词） |
| :--- | :--- |
| `Well-formed hands`（手部完好） | `No extra fingers` |
| `Stable object geometry`（几何稳定） | `No warping` |
| `Natural facial features`（面部自然） | `No face distortion` |
| `Steady motion`（运动稳定） | `No jitter` |
| `Crystal-clear`（画面清晰锐利） | `No blur` |
| `Sharp focus on product`（产品清晰对焦） | `Don't make it blurry` |

### 6.4 版本号

每条提示词末尾自动附加版本号（如 `PROMPT_v1_产品_钩子_日期`），便于校准追踪。


## 7. 预评估：三层快速诊断

> **目的**：替代原五维 100 分制，减少评估 Token 消耗，保留核心风控。

| 层级 | 检查项 | 不通过动作 |
| :--- | :--- | :--- |
| 🔴 **致命层** | 钩子是否有视觉冲突？前 3 秒是否有声音钩子？ | **禁止提交**，退回阶段 2 |
| 🟡 **核心层** | 垂直信号是否前 5 秒口述+字幕双出现？原生感是否启用？ | **警告 + 微调**，用户确认后提交 |
| 🟢 **卫生层** | 是否含否定式负向词？是否超过 100 词/段？ | **自动修正**，直接提交 |

**输出格式**：
- ✅ 三层全通过：可直接提交。
- ⚠️ 核心层不通过：警告 + 建议，用户确认后提交。
- ❌ 致命层不通过：禁止提交，退回阶段 2。
