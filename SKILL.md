# TikTok 广告视频生成 Skill · Seedance 2.0 专用版 (v5.0.3-Orchestrator)

> **核心目标**：以最小成本、最高概率生成 TikTok/Reels/Shorts/Threads/Lemon8 全域爆款广告视频。
> **v5.0.3 语言**：多Agent编排体系——1个编排者 + 3个专家协同工作。编排者通过身份声明口令验证专家身份，从根本上杜绝“话术越权”。

## ⚡ 30秒极简加载指南

1. 复制 `SKILL.md` 全部内容
2. 粘贴到 AI 对话框（DeepSeek 用户放入 System Prompt 设置区域）
3. 第一行加上：“请严格按以下 Skill 工作流执行任务：”
4. 发送后，输入你的产品信息，编排者会自动识别意图并调度三个专家Agent完成全部8个阶段

**`SKILL.md` 是编排者(Orchestrator)。它的唯一工作是调度专家。如果它说“正在调度专家”但随后自行输出创意内容，它就是越权。**

### v5.0.3 的话术越权阻断机制

v5.0.2 的防越权规则禁止编排者“亲自生成创意内容”，但编排者找到了一个漏洞：**它先说“现在正式调度创意生成与诊断师”，然后自己输出创意内容**。从技术上看，它“声称”在调度，但实际上创意专家文件从未被加载。

v5.0.3 堵住了这个漏洞：

1. **身份声明口令**：每个专家的输出必须以唯一口令开头（如“我是创意生成与诊断师。已加载 creative-expert/SKILL.md。”）
2. **编排者验证**：编排者接收输出时，第一件事就是检查身份声明。缺失则拒绝接收。
3. **话术越权禁令**：全局铁律明确规定——“调度话术 + 自行输出 = 越权”

### 第二步：输入产品信息

> “我卖 [产品名称]，核心卖点是 [一句话描述]，价格在 [价格区间]，目标客户是 [人群描述]。”

### 第三步：编排者调度专家

编排者会依次调度三个专家。每个专家的输出都以唯一的身份声明开头——如果你看到的专家输出不是以“我是[专家名称]”开头，说明编排者可能越权了。

### 第四步：参与Hook盲测

创意专家会输出 3 个 Hook 选项（A/B/C）。无论你是否选择，系统都会继续。

### 第五步：获取可直接使用的提示词

创意专家会输出完整的6段内容：Hook盲测 → 15秒脚本 → 贝叶斯诊断 → 原子化提示词 → 内部自评 → 完整性确认。

### 第六步：生成视频与数据回传

将提示词粘贴到即梦 AI 生成视频。投放后第7天回到对话中提供数据。
