## 示例 1: 铸铁元宝锅 · 高冲突型（美国市场）

### 用户输入
> "我卖铸铁元宝锅，核心卖点是物理不粘，无涂层，价格 $29.9，目标客户是 25-45 岁家庭烹饪用户。"

### Hook 盲测选项（Thompson Sampling 生成）
- **A**: "这不是魔术，这是物理！"（冲突型，当前 Beta 参数高）
- **B**: "用了 20 年铁锅，今天才知道开锅是错的"（好奇缺口型）
- **C**: "$29.9 的价格，$299 的质感"（价格冲突型）

*用户选择 A*

### 15 秒脚本输出
```
文案: 这不是魔术

镜头1 (0-3s) | 特写·静态+微颤 | 一颗生鸡蛋落在烧热的铁锅表面，蛋液像水银一样滑动，油脂滋滋声放大，侧逆光泛起金色光泽 | 口播:"看好了"
镜头2 (3-8s) | 近景·慢摇 | 手部持铁铲在锅面划"8"字，正面镜头展示无划痕，字幕"物理不粘" | 口播:"不是涂层"
镜头3 (8-12s)| 中景·环绕 | 连续煎3个鸡蛋，展示每个翻面瞬间，食材在锅内自由滑动 | 口播:"29900次测试，依然如新"
镜头4 (12-15s)| 近景·推拉 | 锅铲入画未接触锅面即定格，购物车图标+字幕"试试看" | 口播:"链接在下面"
```

```

【镜头2】近景, 慢摇, 一只手持金属铲在锅面流畅划出"8"字形, 正面特写展示锅面无任何划痕, 字幕"物理不粘·无涂层"出现在画面下方, 暖色柔光, 4K画质, Seedance 2.0专用

【镜头3】中景, 缓慢环绕, 连续将3颗鸡蛋打入同一锅中, 纪录式展示每个鸡蛋翻面的完美瞬间, 食材在锅内自由滑动不粘底, 自然光+侧逆光, 4K画质, Seedance 2.0专用

【镜头4】近景, 缓慢推拉, 锅铲从画面右侧伸向锅中未完成的菜品, 在即将接触的瞬间画面定格并轻微缩放, 购物车图标+箭头指向下方, 字幕"前200单减$5·点击", 4K画质, Seedance 2.0专用

## 示例 4: AIGC 合规失败修复对照（教育用途）

### 原始提交（被贝叶斯诊断拦截）
```
【镜头1】一个全CG生成的完美厨房，AI厨师在空中翻炒，产品悬浮展示，霓虹灯光
```

### 诊断拦截报告
```
---
【贝叶斯诊断·预提交拦截】
· P(Viral): 0.22
· 95% CI: [0.08, 0.41]
· 决策: 🛑 禁止提交

致命层诊断:
· ❌ AIGC 合规：100% CG画面将触发AIGC标签，自然流量折损15-30%
· ❌ 真实材质占比：0%（核心层要求≥60%）
· ⚠️ 声音Hook缺失

系统自动建议：
1. 将厨房替换为真实拍摄素材（或拟真度>90%的材质渲染）
2. 厨师改为真实人物手部特写
3. 产品展示改为实拍质感展示
---
```

### 修复后 Prompt（通过诊断）
```
【镜头1】中景, 真实厨房环境拍摄, 木质台面+不锈钢灶具清晰可见, 一只手持铸铁锅平稳翻炒, 食材在锅中滑动发出滋滋声, 暖色柔光+自然窗光混合, 4K画质, Seedance 2.0专用
```

### 修复后贝叶斯诊断
```

## 示例 5: 跨市场分发 · 同一产品不同文化策略

### 巴西市场版本
