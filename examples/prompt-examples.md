## 一、成功案例

### 案例 S-001｜铸铁锅 · 物理不粘视觉化 (2025-Q4)

**产品**: 铸铁元宝锅 ($29.9)  
**市场**: 美国  
**Hook 类型**: 冲突/反常识 (CON-01)  
**核心策略**: 将"物理不粘"从参数表移入前3秒视觉——鸡蛋在无油锅中像水银滑动

**关键数据 (投放后7天)**:
- 播放量: 1.2M
- 前3秒留存率: 78%
- 完播率: 61%
- 分享率: 4.2%
- ROI: 3.8x

**可复用公式**:
```
[特写] + [反常物理现象] + [声音Hook放大] + [3秒内无口播，纯视觉冲击]
```

**完整脚本摘要**:
- 0-3s: 鸡蛋落在烧热铁锅，滑动如液态金属，滋滋声
- 3-8s: 铁铲划"8"字，无划痕
- 8-12s: 连续煎3蛋，翻面流畅
- 12-15s: 锅铲悬停定格 + CTA

**成功要素拆解**:
1. 视觉冲突前3秒出现，无延迟
2. 声音设计三层（油脂声+铲触声+POP音）
3. 复播彩蛋（锅铲未接触即定格）触发 Zeigarnik 效应
4. 无涂层物理不粘的卖点被"可视化"，无需口播解释

**🆕 v4.0.1 回传**:
- 爆款判定: ✅ 是
- 对 Beta(α, β) 贡献: α += 1（冲突型 Hook 的 α 增加）
- P(Viral) 事后估计: 0.78 [0.65, 0.89]（基于本案例及同类案例的后验）
- 动态权重更新: w_hook_conflict 经更新后提升至 1.32（以1.0为基准）
- 步长裁剪状态: 未触发（CTR_ratio 在正常范围）

```

- 前3秒留存率: 70%

```
