**叙事逻辑**：痛点快切 → 过程微距 → 惊艳结果（3-4 镜头）
**原生感策略**：素人真实反应 + 生活化杂乱背景 + 自然窗光

### 示例 1.1：纳米清洁海绵（认知失调钩子·4镜头）

#### 混写版（推荐使用）

```
[原生感策略] 真实素人反应 (Authentic unscripted reaction), 生活化杂乱背景 (Lived-in messy background), 自然窗光 (Natural window light).

[镜头1: 0-3秒]
4K UHD (超高清), 48fps (高帧率), 35mm lens (广角). 快速摇摄 (Whip pan) 扫过做完饭后油腻杂乱的灶台，厚重黄色油垢清晰可见。手持第一人称视角 (Handheld POV)，轻微呼吸感 (Subtle breathing)。干净明亮布光 (Clean bright lighting)。干脆利落的运镜 (Snappy motion)。

[镜头2: 3-4秒]
4K UHD, 48fps, 100mm macro lens (微距镜头). 快速切入 (Quick cut) 蓝色清洁海绵接触油污的极近特写。油垢表面有通透高光 (Crisp highlights)。固定镜头 (Static shot)。

[镜头3: 4-7秒]
4K UHD, 120fps (高帧率), 100mm macro lens (微距镜头). 极近微距慢动作 (Extreme macro slow motion)：油污接触瞬间消失，露出镜面般反光的不锈钢底层。围绕洁净点轻微环绕 (Slow orbit)。画面随清脆“叮”声同步 (Visual beats sync with crisp "ding")。无形变，物体几何稳定 (No warping, Stable object geometry)。

[镜头4: 7-15秒]
4K UHD, 50mm lens. 缓慢拉远 (Slow dolly-out) 至整个灶台完美洁净的全景。手在反光表面滑动，无任何残留。定格于镜面般反光灶台2秒 (Freeze frame for 2 seconds)，留有文字空间：“这怎么可能？👇”。左下角留出“Save for your next deep clean”文字空间。右下角留出“Real cleaners know this hack 🧽”文字空间。柔光箱布光 (Softbox lighting)，4K超高清，9:16竖屏 (9:16 vertical)，15秒无缝循环。手部完好无多余手指 (Well-formed hands, No extra fingers)。评论引导时刻，分享结尾。
```

## 九、提示词质量自检清单（v2.6 更新）

**五维架构检查**
- [ ] 是否采用了混写格式（中文意境 + 英文精准指令）？
- [ ] 是否明确标注了每个镜头的技术基底（4K UHD, 48fps, 镜头焦段）？
- [ ] 是否明确标注了每个镜头的镜头运动（推/拉/摇/移/跟/环绕）？
- [ ] 是否精确描述了视觉元素？
- [ ] 是否独立指定了光影系统？
- [ ] 是否加入了动态设计（转场方式、音频同步、定格）？

**原生感策略检查（新增）**
- [ ] 是否强制融入了原生感词汇（真实素人反应、生活化背景、自然窗光）？
- [ ] 是否避免了精修广告的精致感？

**多镜头结构检查**
- [ ] 是否明确标注了每个镜头的时长（如 [0-4秒]）？
- [ ] 镜头数量是否与品类模板匹配（3-4 个）？
- [ ] 转场方式是否明确且适合内容？

**互动策略检查**
- [ ] 是否包含了收藏引导的文字空间或话术？
- [ ] 是否包含了社交货币分享引导（展现品味/展示专业/圈层归属）？

**去AI味检查**
- [ ] 是否包含 `手部完好无多余手指`（如有人物手部）？
- [ ] 是否包含 `无脸部变形，面部自然`（如有人物面部）？
- [ ] 是否避免使用负面词表中的词汇？

**成本控制检查**
- [ ] 测试阶段是否推荐使用 Fast 模式精简版？
