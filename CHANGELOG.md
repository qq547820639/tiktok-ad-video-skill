# 版本变更日志 (CHANGELOG)

## [v2.8] - 2026-05-20

### 🚀 新增
- **声音钩子优先原则**：在 `SKILL.md` 核心铁律中新增“前 3 秒优先使用声音钩子（ASMR/音效/环境音），口播第 3 秒后才进入”。用户平均 2.8 秒就划走，声音比人声穿透力更强。
- **品类钩子选型对照表（实战版）**：在 `viral-hook-patterns.md` 中新增 §二，基于 80+ 条视频实战数据，明确香水/美妆、锅具/厨房、3C/家电、鞋服/百货、清洁用品等品类的推荐钩子、避免钩子、前 3 秒核心动作及声音策略。
- **数据驱动迭代指南**：新增独立文档 `references/data-driven-iteration.md`，含核心指标健康基准、诊断决策树、分数据表现迭代动作表、品类专属迭代建议、长期迭代节奏。
- **发布前自查清单**：新增独立文档 `references/self-check-checklist.md`，可打印逐项核对：前 3 秒钩子检查、品类钩子选型检查、脚本结构检查、字幕大字报检查、发布前最后确认、数据反馈迭代动作。
- **声音钩子执行度评分项**：在 `evaluation-rubric.md` 导演执行维度中新增“声音钩子执行度”子项（3 分），评估前 3 秒声音策略及口播进入时机。
- **长期内容矩阵建议**：在 `SKILL.md` 和 `self-check-checklist.md` 中强调每个产品至少准备 3 条不同钩子方向视频同时跑（情绪钩子 + 功能钩子 + 性价比钩子）。

### 🔄 变更
- **多镜头模板融入声音策略**：`viral-hook-patterns.md` §四所有品类多镜头模板增加“声音策略”列，明确前 3 秒声音钩子类型。
- **各钩子完整脚本融入声音钩子**：`viral-hook-patterns.md` §五所有脚本示例增加声音策略描述（如清洁海绵用 ASMR 擦拭声，铸铁锅用 ASMR 无油煎蛋声）。
- **SKILL.md 工作流升级**：阶段 2 脚本示例增加声音策略列，阶段 3 提示词工程融入声音钩子优先原则，阶段 4 分发策略增加声音相关提示。
- **评分表权重微调**：`evaluation-rubric.md` 中五维架构执行度调整为 2 分，原生感执行度调整为 2 分，新增声音钩子执行度 3 分，垂直领域信号执行度调整为 2 分，多镜头结构/音频同步/转场质量各调整为 2 分。
- **前 3 秒留存评分标准更新**：明确要求前 3 秒有声音钩子且口播第 3 秒后才进入才能获得 9-10 分。
- **失败模式扩充**：新增“钩子类型选错”“声音钩子缺失”两条失败模式及修复方案。
