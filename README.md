# TikTok 广告视频生成 Skill · Seedance 2.0 专用版 (v5.0.2-Orchestrator)

> **核心目标**：以最小成本、最高概率生成 TikTok/Reels/Shorts/Threads/Lemon8 全域爆款广告视频。
> **v5.0.2 语言**：多Agent编排体系——1个编排者 + 3个专家协同工作。创意专家内置执行触发器、阶段5前置检查和修正清单全链路追溯，从根本上杜绝越权执行和阶段跳过。

## ⚡ 30秒极简加载指南

1. 复制 `SKILL.md` 全部内容
2. 粘贴到 AI 对话框（DeepSeek 用户放入 System Prompt 设置区域）
3. 第一行加上：“请严格按以下 Skill 工作流执行任务：”
4. 发送后，输入你的产品信息，编排者会自动识别意图并调度三个专家Agent完成全部8个阶段

**`SKILL.md` 是编排者(Orchestrator)。它不会亲自生成创意，而是用【系统指令·不可协商】格式调度三个专家Agent分工协作。**

## 架构概览：多Agent编排体系 + 三重保障

    ▼

│  · 发送【系统指令·不可协商】 │
│  · 完整性检查 + 越权阻断     │

└─────────────────────────────┘
    │
    ├─→ 专家1: 趋势与竞品分析师
    │    └─ 阶段1-2: 趋势扫描 + 竞品拆解

    ├─→ 专家2: 创意生成与诊断师 (核心)
    │    └─ 阶段3-6.5: Hook→脚本→诊断→提示词→自评
    │         ├─ §〇.5 执行触发器（6条无条件规则）
    │         ├─ §四 阶段5前置检查（5项缺失则阻断进入阶段6）

### v5.0.2 解决的核心问题

| 已发现的漏洞 | v5.0.2 修正 |
|:---|:---|
| 编排者越权：自行生成脚本和诊断，跳过专家调度 | 全局铁律第1条增强：禁止自行判断“用户需求简单”而跳过专家。越权时必须回复指定话术并立即调度专家。 |
| 创意专家跳过阶段5：直接输出提示词，无P(Viral) | 阶段5前置检查：进入阶段6前强制验证P(Viral)+CI+决策建议+修正清单五项是否完整。缺失则阻断。 |
| 阶段5修正清单与阶段6/6.5脱节：修正建议无人落实 | 修正清单全链路追溯：阶段5修正清单→阶段6修正对照表→阶段6.5修正清单追溯，三项必须一一对应。 |
| 用户意图被降级：完整生成被误判为快速模式 | §〇 用户意图识别：明确完整生成的判定关键词，严禁自行降级。 |

### 三重保障机制

**第一重（创意专家内部·事前阻断）**：
- 阶段5前置检查：进入阶段6之前，验证P(Viral)+CI+决策建议+修正清单五项完整。缺失则不得进入阶段6。

**第二重（创意专家内部·事后自检）**：
- 自检熔断：输出末尾强制检查6项完整性，缺失则立即补充。
- 修正清单全链路追溯：阶段5→6→6.5的修正建议必须形成闭环。

**第三重（编排者外部·接收检查）**：
- 完整性检查：接收创意专家输出后，检查是否包含全部6段内容。缺失则拒绝接收并要求重做。
- 防越权阻断：编排者自己绝不生成创意内容。若用户直接要求，必须回复指定话术并调度专家。

### 第二步：输入产品信息

向加载了编排者 SKILL 的 AI 助手发送你的产品信息：

> “我卖 [产品名称]，核心卖点是 [一句话描述]，价格在 [价格区间]，目标客户是 [人群描述]。”

**示例**：

> “我卖铸铁元宝锅，无涂层物理不粘，价格 $29.9，目标客户是 25-45 岁家庭烹饪用户。”

### 第三步：编排者自动识别意图并调度专家

编排者会首先执行意图识别，然后依次调度三个专家：

1. **趋势与竞品分析师** → 趋势信号 + 竞品拆解
2. **创意生成与诊断师** → Hook盲测 + 脚本 + 诊断(P(Viral)+CI) + 提示词 + 自评
3. **发布与数据顾问** → 平台分发建议 + 数据回传模板

### 第四步：参与Hook盲测

创意专家会输出 3 个 Hook 选项（A/B/C），你可以选择，也可以不选（系统默认采用最优臂继续）。**无论你是否选择，系统都会继续执行后续阶段。**

### 第五步：获取可直接使用的提示词

创意专家会输出：
1. **贝叶斯诊断摘要**（P(Viral) + Bootstrap CI + 修正清单）
2. **4段Seedance 2.0原子化提示词**（含修正对照表——明确标注每一项修正的来源和落实位置）
3. **内部自评记录**（含修正清单追溯——确认阶段5的每项修正都在阶段6中落实）
4. **完整性确认清单**（6个阶段的完成状态）

### 第六步：生成视频与数据回传

1. 将提示词逐段粘贴到即梦 AI 生成视频
2. 投放后第7天回到对话中提供数据
3. 编排者会识别为"数据回传"意图，调度创意专家执行权重更新
