## 🔄 自我迭代机制

Skill 具备数据驱动的自进化能力：

1. **每次任务** → 后台生成《自检报告》
2. **每个产品** → 填写 `product-tracker-template.md` 归因分析
3. **连续 3 次验证** → 触发钩子权重调整
4. **发现失败模式** → 更新词汇表或负面词库
