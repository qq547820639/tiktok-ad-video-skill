## 1. 核心铁律（精简版）

1. **15 秒即全部**：所有脚本、钩子、转化路径必须在 15 秒内闭环。
2. **前 3 秒定生死**：首帧必须包含视觉冲突/悬念。**前 3 秒优先使用声音钩子（ASMR/音效），口播第 3 秒后才进入**。
3. **原生感优先**：视频应看起来像真实用户分享（UGC风格），使用素人演员、自然光、手持感。
4. **垂直领域信号**：前 5 秒口述+字幕双重强化核心主题词（如 "cleaning hack"）。
5. **品类匹配钩子**：对照下表选择推荐钩子，避免选错（如香水用价格对比）。
6. **收藏+分享优先**：互动引导优先设计收藏和社交货币分享话术。
7. **先验证、后投入**：提交生成前必须通过 Prompt 预评估（≥80分）。

## 3. 工作流（6 阶段精简版）

### 阶段 0：选品与品类匹配
1. 提取产品卖点，判断品类，对照上表确定推荐钩子。
2. 确定核心主题词（1-2个），强制启用原生感。

### 阶段 1：钩子文案三选一（零成本盲测）
输出 3 个钩子选项，用户盲选。示例：
```
A. [认知失调型]：别再用普通抹布了！你擦掉的只是水，不是细菌。
B. [极简结果型]：3秒见证厨房台面从油腻到反光。
C. [价格锚点型]：不是几百块的清洁服务请不起，而是9.9的它更有性价比。
```

### 阶段 2：15 秒多镜头脚本
调用 4 镜头模板（功能效果型）：
- **0-3s**：痛点快切（ASMR/音效优先，无口播）
- **3-7s**：过程微距（产品介入瞬间，口播进入）
- **7-12s**：过程持续（多角度快切）
- **12-15s**：结果定格 + 收藏引导 ("Save for next ___") + 社交货币分享

### 阶段 3：五维提示词工程（混写推荐）
按五维架构输出提示词：`[技术基底] + [原生感词汇] + [镜头描述] + [声音钩子] + [去AI味指令]`。

### 阶段 3.5：Prompt 预评估（积分风控闸门）
五维度打分（前3秒钩子30分/声音策略25分/垂直信号20分/原生感15分/去AI味10分）：
- **≥80分**：可提交即梦
- **<80分**：退回阶段2优化，不消耗积分

### 阶段 4：多平台分发
提醒 AIGC 标签（TikTok: AI-generated, Meta: Made with AI），输出各平台标题/标签/互动话术。

### 阶段 6：数据迭代
发布 7 天后反馈数据，AI 根据完播率/收藏率判断迭代方向。


## 4. 成本控制

- 测试阶段用 **Fast 模式**（约 60-84 积分/次）
- 必须先通过预评估才可提交生成
- 正式投放用 Standard 模式（120 积分/次）


## 5. 常用指令速查

**声音钩子**：`前3秒纯 ASMR 音效` / `Visual beats sync with heavy bass`

**原生感词汇**：`Authentic unscripted reaction` / `Lived-in messy background` / `Natural window light`

**去AI味**：`Well-formed hands, No extra fingers` / `Avoid slow cinematic pans`

**平台尾缀**：`9:16 vertical, 15 seconds loop-ready`


**记住**：前 3 秒声音钩子比画面更重要；钩子选错满盘皆输；预评估不通过绝不消耗积分。
